## Full Assembly Test Order

Follow this order. Stop if any joint risks cracking.

1. Legs to pelvis.
2. Tail to pelvis.
3. Torso to pelvis.
4. Arms to torso.

7. Route-reader in hand.

10. Neutral standing pose.

## Go / No-Go Decision for Paint Prototype

### Go to Paint if:

- [ ] Full prototype assembles without cracking.
- [ ] Figure stands without required display base.
- [ ] Tail supports balance and character.
- [ ] Head expression reads correctly.
- [ ] Hands hold selected accessories.
- [ ] All required canon markers are visible.
- [ ] Paint zones are accessible.
- [ ] No required part is too fragile.
- [ ] Paint-rub samples pass or masking plan exists.

### Conditional Go if:

- [ ] Minor sanding fixes fit.
- [ ] Paint masking is required for joints.
- [ ] One accessory needs small thickness adjustment.
- [ ] Tail curve needs minor change but current version stands.

### No-Go if:

- [ ] Figure cannot assemble safely.
- [ ] Figure cannot stand.
- [ ] Tail socket or pelvis fails.
- [ ] Head read is wrong.
- [ ] Hands cannot hold accessories.
- [ ] Required canon markers are missing.
- [ ] Multiple structural parts require redesign.

Decision:

- [ ] Go to paint
- [ ] Conditional Go to paint
- [ ] No-Go, revise and reprint
