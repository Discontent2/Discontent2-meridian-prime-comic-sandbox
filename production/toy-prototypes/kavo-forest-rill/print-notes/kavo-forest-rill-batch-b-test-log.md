## Purpose

This file logs the results of Batch B for the Kavo "Forest" Rill action figure prototype.

Batch B tests the first partial body assembly: torso, pelvis, legs, arms, keyed tail, stance, balance, seam behavior, and whether the sculpt still reads as Kavo once it becomes more than coupons and loose gear.

Use this file after completing Batch A and selecting the required tolerances.
