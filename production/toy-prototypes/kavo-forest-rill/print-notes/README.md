# Print Notes

Use this folder for 3D print production notes for the Kavo "Forest" Rill prototype.

Appropriate contents:

- resin and printer settings
- orientation notes
- support notes
- joint tolerance tests
- failed print notes
- sanding and curing observations
- tail balance tests
- accessory durability tests
- STL / 3MF export notes

Keep safety notes visible whenever resin, sanding, sharp parts, or brittle test pieces are discussed.
