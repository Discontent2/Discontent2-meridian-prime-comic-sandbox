# Paint Guide

Use this folder for production paint notes for the Kavo "Forest" Rill prototype.

Appropriate contents:

- color swatch tests
- primer notes
- skin palette decisions
- red proto-quill paint tests
- checker trim placement tests
- lens finish tests
- badge paint notes
- weathering passes
- sealant tests
- final paint recipe

Paint choices should preserve canon markers while supporting shelf read and 3D printed prototype durability.
