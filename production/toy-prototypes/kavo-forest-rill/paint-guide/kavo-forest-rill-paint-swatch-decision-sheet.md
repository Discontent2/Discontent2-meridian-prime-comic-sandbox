## Purpose

This file records the approved Prototype A paint decisions for the first painted Kavo "Forest" Rill prototype.

The goal is to lock the first palette before model-sheet art, sculpt review, print testing, and painted prototype work begin.

---

## Paint Philosophy

Kavo's paint should read as:

- Velocisapien / Velociraptoroid
- pressure-route street culture
- Mountaineer / GPR path-reader
- cold industrial field professional
- relaxed menace with useful gear
- retro toyetic, but not neon parody

The paint should make the sculpt easier to read, not drown it in decorative static.

## Gear Weathering Tests

| Level | Description | Use Case | Approval |

| W-04 | heavy route damage | Not recommended for first prototype. | [ ] Not selected |

Approved W-02 weathering zones:

- lower feet
- lower tail contact point
- harness edges
- route-reader corners
- EMT pouch flap
- route probe grip
- black flag lower pole

Avoid:

- random blood
- heavy mud covering silhouette
- rust on everything
- face-obscuring grime
