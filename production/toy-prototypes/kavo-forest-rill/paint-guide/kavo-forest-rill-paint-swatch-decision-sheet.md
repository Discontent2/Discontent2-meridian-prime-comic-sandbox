**Room:** Meridian Prime Toy Development Room / Action Figure Prototype Lab  
**Folder:** `production/toy-prototypes/kavo-forest-rill/paint-guide/`  
**Status:** First-pass paint swatch and approval sheet  
**Prototype Target:** 5-inch retro collector action figure  
**Canon Boundary:** Sandbox production file. Does not alter main canon.

---

## Purpose

This file defines candidate color swatches, canon-required color markers, toy-only paint variants, weathering tests, lens finish tests, and approval boxes for the first painted Kavo "Forest" Rill prototype.

The goal is to make paint decisions before the figure becomes a tiny saurian argument with primer on it.

---

## Paint Philosophy

Kavo's paint should read as:

- Velocisapien / Velociraptoroid
- pressure-route street culture
- Mountaineer / GPR path-reader
- cold industrial field professional
- relaxed menace with useful gear
- retro toyetic, but not neon parody

The paint should make the sculpt easier to read, not drown it in decorative static.

## Checker Trim Placement Tests

Checker trim should be subtle, not costume-dominant.

Candidate placements:

| Placement | Description | Risk | Approval |
|---|---|---|---|
| CT-01 | one harness tab | Safest, very subtle. | [ ] |
| CT-02 | jacket / vest cuff edge | Visible but still restrained. | [ ] |
| CT-03 | small side stripe on garment | Stronger toy read. | [ ] |
| CT-04 | badge-adjacent patch | Easy to miss at shelf distance. | [ ] |

Recommended first prototype: CT-01 plus CT-02, if sculpt supports it.

Do not use all-over checkerboard.

## Gear Weathering Tests

Weathering should show use, not neglect.

Test levels:

| Level | Description | Use Case | Approval |
|---|---|---|---|
| W-01 | clean prototype paint | Sculpt review, packaging mockup. | [ ] |
| W-02 | light route dust | Recommended first painted prototype. | [ ] |
| W-03 | moderate field wear | Later story-heavy variant. | [ ] |
| W-04 | heavy route damage | Not recommended for first prototype. | [ ] |

Recommended: W-02.

Weathering zones:

- lower feet
- lower tail contact point
- harness edges
- route-reader corners
- EMT pouch flap
- route probe grip
- black flag lower pole

Avoid:

- random blood
- heavy mud covering silhouette
- rust on everything
- face-obscuring grime
