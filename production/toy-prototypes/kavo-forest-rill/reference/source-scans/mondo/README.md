# Mondo scan source package

Place the toy scan package here:

```text
production/toy-prototypes/kavo-forest-rill/reference/source-scans/mondo/Mondo.zip
```

Expected contents:

```text
model.glb
tex_u0_v0_diffuse.jpg
tex_u0_v0_normal.jpg
```

After the package is in this folder and the repo is pulled locally, run:

```text
production/toy-prototypes/kavo-forest-rill/reference/blockout-submissions/kfr-blockout-v004/kfr_mondo_scan_reference_import_001.py
```

The import script brings the GLB into Blender as a scaled reference mannequin for Kavo Forest Rill.
