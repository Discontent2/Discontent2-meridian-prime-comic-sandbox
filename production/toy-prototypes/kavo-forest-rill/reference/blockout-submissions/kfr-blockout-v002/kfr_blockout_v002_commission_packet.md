## Required Improvements Over v001

### 1. Overall Mesh

Replace primitive block shapes with sculpt-aware masses.

The model may still be low-detail, but forms should feel intentionally sculpted and printable, not assembled from default geometry toys that escaped the drawer.

### 2. Head Identity

Refine:

- saurian snout length
- brow weight
- mouth line placement
- shades mass
- short sparse proto-quill mass
- neck base
- calm-dangerous Forest expression

Do not add fine scale texture yet.

### 3. Body and Garments

Refine:

- cropped route jacket shape
- sturdy collar mass
- jacket cuffs
- black field-knit under-layer area
- harness path over garment layers
- cargo pants shape
- thigh / knee reinforcement panels
- cargo pocket placement
- pant cuff to boot collar transition

Keep all folds broad and toy-readable.

### 4. Arms, Hands, and Watch

Refine:

- four-finger hands
- black fingerless glove mass
- grip hand shape
- wrist swivel placeholder
- compact left-wrist GPR watch

Watch must not become a wrist warmer, gauntlet, cuff, or giant outdoor GPS brick.

### 5. Legs, Boots, and Stance

Refine:

- broad boot footprint
- four-toe claw clearance
- boot collar shape
- stable hip width
- optional knee seam logic if preserved

No functional ankle articulation for Prototype A.

### 6. Tail and Socket

Refine:

- thick tail base
- keyed socket volume
- taper
- blunt tip
- balance path
- tail clearance from jacket hem and harness

Tail should feel expressive and useful, not decorative.

### 7. Joint Placeholders

Replace marker spheres with sculpt-aware placeholders:

- neck
- shoulders
- wrists
- hips
- optional waist
- optional knees
- tail keyed socket

Do not fully engineer final toleranced joints yet. Make them clear enough to derive coupon geometry after v002 approval.

---

## Hard Guardrails

Reject the blockout if it includes:

- human skin on Forest
- human facial structure
- wrist warmers
- oversized watch
- watch on wrong wrist
- ninja / assassin read
- military operator read
- dinosaur monster posture
- generic raptor scout read
- real-world watch / boot / garment branding
- weapons-first accessories
- fragile dangling straps or cords
- tiny final-detail noise used to hide bad proportions
