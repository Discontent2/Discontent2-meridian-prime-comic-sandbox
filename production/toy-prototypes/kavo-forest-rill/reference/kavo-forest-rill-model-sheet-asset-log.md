## Purpose

This file tracks future model-sheet images, prompt outputs, artist reference files, packaging reference images, paint references, and sculpt-support visuals for the Kavo "Forest" Rill action figure prototype.

Use this log to record where each asset came from, what prompt or artist note generated it, whether it is canon-safe, whether it has been approved, and what production use it supports.

No image, sketch, render, or prompt output should become sculpt reference by accident. Tiny plastic dinosaurs deserve paperwork too.

---

## Asset Status Definitions

### Proposed

Asset exists or is planned, but has not been reviewed.

### Review Needed

Asset is promising but needs canon, toy, printability, or originality review.

### Approved for Sculpt Reference

Asset may guide sculpt shape, silhouette, anatomy, gear placement, or turnaround views.

### Approved for Paint Reference

Asset may guide color, finish, weathering, lens treatment, or material breakup.

### Approved for Packaging Reference

Asset may guide cardback, portrait art, accessory callouts, or presentation mood.

### Archive Only

Asset should be kept for context but not used to guide production.

### Rejected / Do Not Use

Asset conflicts with canon, originality, printability, safety, or toy-development guardrails.

---

## Canon-Safe Review Questions

For each asset, ask:

- Does Kavo read as Velocisapien / Velociraptoroid?
- Does he read as Mountaineer / route guide / wilderness EMT / GPR path-reader?
- Are the tail, four-fingered hands, and four-toed feet preserved?
- Are red proto-quills, shades / goggles, harness, checker trim, badges, and claw-wraps present or planned?
- Does the posture read relaxed route-readiness rather than attack pose?
- Does it avoid ninja, assassin, monster, generic raptor scout, human reptile makeup, and protected-toyline clone cues?
- Are toy-only details labeled properly?
- Are candidate canon details logged before use?

## Approval Checklist for Sculpt Reference

An asset can be marked **Approved for Sculpt Reference** only if:

- [ ] full body proportions are consistent with the 5-inch figure plan
- [ ] Kavo reads as Velocisapien / Velociraptoroid
- [ ] required anatomy markers are present
- [ ] tail is functional and characterful
- [ ] head expression fits relaxed menace
- [ ] hands and feet match four-finger / four-toe canon
- [ ] harness, badges, wraps, and shades / goggles are clear
- [ ] no protected-toyline confusion is present
- [ ] details are printable or marked for simplification
- [ ] toy-only or candidate canon details are labeled

---

## Approval Checklist for Paint Reference

An asset can be marked **Approved for Paint Reference** only if:

- [ ] red proto-quill mohawk is preserved
- [ ] skin palette is compatible with the swatch decision sheet
- [ ] shades / goggles finish is clear
- [ ] harness and gear colors read field-practical
- [ ] checker trim is subtle
- [ ] badge colors do not imply unapproved canon meanings
- [ ] weathering is restrained
- [ ] accessory colors match the paint guide
- [ ] figure does not drift into neon parody or monster gloss

---

## Approval Checklist for Packaging Reference

An asset can be marked **Approved for Packaging Reference** only if:

- [ ] it uses original Meridian Prime visual language
- [ ] it avoids protected trade dress
- [ ] it avoids turtle, sewer, pizza, ooze, mutagen, and ninja cues
- [ ] it preserves Kavo's role and character read
- [ ] it does not reveal hidden Book One information
- [ ] it includes adult collectible prototype caution if used in mock packaging
