## Canon-Safe Core Description

Kavo "Forest" Rill is a Velocisapien / Velociraptoroid Mountaineer, route guide, wilderness EMT, and GPR path-reader from Meridian Prime. He has a mostly humanoid saurian body plan, expressive saurian face with snout and brow structure, scaled skin, four-fingered clawed hands, four-toed clawed feet, and a universal tail used for balance, attitude, combat posture, and emotion. He has a short sparse red proto-quill mohawk, black-frame shades when not wearing route goggles, subtle checker trim, a light route harness, pressure-route badges, claw-wraps, and a relaxed posture that can become dangerous instantly.

He should feel like relaxed menace: precise, streetwise, impossible to rush, quiet until funny, funny until dangerous, dangerous until useful.

## Master Model Sheet Prompt

Use this as the main concept prompt:

```text
Toy Development Concept, Sandbox Only. Create a clean sculptor-ready model sheet for Kavo "Forest" Rill, a Velocisapien / Velociraptoroid Mountaineer and GPR path-reader from the Meridian Prime universe, designed as a 5-inch retro collector action figure prototype. Show front, side, and back views on a neutral background with consistent proportions. He has a mostly humanoid saurian body plan, expressive saurian face with snout and brow structure, scaled skin, four-fingered clawed hands, four-toed clawed feet, and a long tail engineered for balance and attitude. He has a short sparse red proto-quill mohawk, black-frame shades, optional route goggles, subtle checker trim, light route harness, pressure-route badge cluster, claw-wraps, and practical Mountaineer field gear. His posture is relaxed route-readiness, calm and sly, dangerous only when needed. Chunky retro toyetic proportions, strong silhouette, durable 3D-printable shapes, broad readable details, no fragile tiny parts. Include accessory callouts for a GPR / route-reader device, black Path flag marker, route probe, compact EMT pouch, and route goggles. Do not make him a ninja, assassin, generic raptor scout, dinosaur monster, animalistic alien, human in reptile makeup, or clone of any existing toyline. No turtle shell, no mask, no sewer, no pizza, no ooze, no mutagen, no ninja vocabulary, no literal music props.
```

## Turnaround Prompt

```text
Toy Development Concept, Sandbox Only. Create a front / side / back turnaround sheet for Kavo "Forest" Rill as a 5-inch retro collector action figure prototype. Keep the same proportions across all views. Emphasize a mostly humanoid saurian Velocisapien body plan, expressive snout and brow, four-fingered clawed hands, four-toed clawed feet, scaled skin, short sparse red proto-quill mohawk, black-frame shades, light route harness, subtle checker trim, pressure-route badge cluster, claw-wraps, and a sturdy tail shaped for balance. Pose should be neutral engineering stance with relaxed route-readiness, not sprinting or fighting. Show tail connection, harness path, badge placement, pouch placement, and foot stance clearly. Chunky retro toyetic proportions, durable print-friendly shapes, no tiny fragile slivers.
```

## Head and Expression Sheet Prompt

```text
Toy Development Concept, Sandbox Only. Create a head and expression sheet for Kavo "Forest" Rill, a Velocisapien / Velociraptoroid Mountaineer. Show front, side, and three-quarter head views. ## Accessory Sheet Prompt
```

```text
Toy Development Concept, Sandbox Only. Create an accessory sheet for Kavo "Forest" Rill's 5-inch retro action figure prototype. Show each accessory scaled beside a four-fingered clawed hand: GPR / route-reader device, black Path flag marker, route probe / collapsible field staff, compact EMT pouch, route goggles, and optional brass whistle or route token sculpted as harness detail. Accessories should be practical Mountaineer and Traverse field tools, not fantasy weapons. Make handles thick enough for 3D printing and compatible with gripping hands. The black Path flag marker should have a thickened pole and matte black flag. The route probe should have a blunt tool tip, not a spear point.
```

---

## Articulation Overlay Prompt

```text
Toy Development Concept, Sandbox Only. Create an articulation overlay for a 5-inch Kavo "Forest" Rill retro action figure prototype. Show joint points clearly on a neutral full-body view: neck swivel or simple ball, shoulder swivels, wrist swivels, simple hip joints, keyed removable or swivel-ready tail, optional waist, optional future elbows and knees. Indicate seams hidden by wrist wraps, harness straps, scale transitions, belt line, and tail base folds. Prioritize durable prototype engineering over extreme poseability. The figure should support relaxed route-readiness, head turn, accessory holding, stable feet, and tail-assisted balance.
```

## Model Sheet Acceptance Checklist

Before moving to sculpt:

- [ ] Front, side, and back views match proportions.
- [ ] Kavo reads as Velocisapien / Velociraptoroid.
- [ ] Kavo reads as Mountaineer / route guide / wilderness EMT / GPR path-reader.
- [ ] Tail is essential and functional, not decorative.
- [ ] Hands have four fingers and can hold accessories.
- [ ] Feet have four toes and can support the figure.
- [ ] Shades / goggles are clear but printable.
- [ ] Proto-quill mohawk is short, sparse, and red.
- [ ] Checker trim is subtle.
- [ ] Gear is practical and canon-based.
- [ ] Figure does not read as ninja, assassin, generic raptor scout, dinosaur monster, or protected-toyline clone.
- [ ] Accessories are scaled for 3D print handling.
- [ ] Articulation overlay matches the parts breakdown plan.
- [ ] Paint-zone overlay matches paint swatch decision sheet.
