This packet translates the approved Kavo "Forest" Rill toy-development references into sculptor-ready parts, joints, engineering notes, accessory compatibility rules, and first-print build priorities.

Use this file before blocking out the Blender / ZBrush / Nomad Sculpt model.

## Build Philosophy

Prototype A should prioritize:

1. strong silhouette
2. durable printability
3. recognizable Kavo identity
4. stable standing pose
5. useful tail balance
6. accessory handling
7. clean paint masking
8. simple reliable joints

A durable prototype is better than a tiny joint carnival that shatters when the tail side-eyes gravity.

## Figure Scale and Proportion Target

**Scale:** 5-inch retro collector figure  
**Approximate height:** 5 inches / 127 mm to top of head or proto-quill silhouette  
**Proportion language:** chunky, toyetic, retro-action, not goofy  
**Pose attitude:** relaxed route-readiness, not mid-sprint

### Proportion Targets

| Area | Sculpt Target |
|---|---|
| Head | Large enough for expressive saurian face, brow, shades, snout, and short sparse red proto-quills. |
| Torso | Chunky enough to hold harness and socket tolerances. Avoid thin heroic waist. |
| Arms | Strong but not bodybuilder. Must hold route tools. |
| Hands | Four fingers per hand, including thumb-equivalent. Gloves must preserve visible saurian fingers and claws. |
| Legs | Stable, field-ready stance. Avoid long sprinting stilts. |
| Feet / boots | Wide enough for freestanding display. Four-toe claw clearance preserved. |
| Tail | Functional balance shape. Thick base, strong taper, blunt tip. |

## Joint Engineering Notes

### Neck

- Use thick peg and sturdy head socket.
- Preserve neck clearance from KFR-REF-005.
- Avoid thin shades arms interfering with head rotation.

### Shoulders

- Shoulder clearance must be checked against harness straps from KFR-REF-010.
- If ball shoulders are too fragile, use robust swivels first.
- Shoulder movement should support route probe, route-reader, and flag marker poses.

### Wrists

- Wrist seam should sit under black glove cuff.
- Wrist pegs must be thick enough to survive repeated fitting.
- Grip hands should match KFR-REF-011 accessory handle diameters.

### Hips

- Use stable retro hips.
- Avoid tiny internal pins that are difficult to cure / clean.
- Confirm legs clear harness lower straps and pouches.

### Knees

- Optional only.
- If included, use a simple hinge or cut joint with generous wall thickness.
- If knee weakens the boot stance, omit it for Prototype A.

### Ankles / Boots

- No functional ankle articulation for Prototype A.
- Use boot collar seam as visual transition only.
- Boots must stand flat and wide.

### Tail

- Tail base must be thick and socketed into pelvis / lower torso.
- Use keyed shape so tail installs only in correct orientation.
- Design socket as swivel-ready only if wall thickness allows.
- Tail tip must be blunt.
- Tail should contribute to balance without becoming a sharp prop.

## Joint Test Coupons

Print small test pieces before full prototype:

- neck peg / socket
- shoulder peg / socket
- wrist peg / socket
- hip peg / socket
- tail keyed peg / socket
- optional knee hinge test
- accessory handle / gripping hand test

Suggested clearance ladder for resin prototypes:

| Test | Clearance |
|---|---:|
| Tight | 0.10 mm |
| Medium | 0.15 mm |
| Loose | 0.20 mm |

Actual tolerances depend on printer, resin, cure time, shrinkage, supports, sanding, primer, and sealant. Do not assume final fit before test coupons are cured and painted.

## Sculpt Detail Requirements

### Head

- Use KFR-REF-005 as primary head reference.
- Strong saurian brow.
- Expressive snout and mouth line.
- Sly, calm, precise expression.
- Thick black-frame shades.
- Short sparse red proto-quills, sturdy and not needle-thin.
- Preserve enough neck clearance for joint movement.

### Body / Torso

- Chunky retro toyetic build.
- Harness integrated cleanly.
- Chest and back panels visible.
- Checker trim zones raised or recessed enough to paint cleanly.
- Badge details should be sculpted as simple readable shapes, not tiny flakes.
- Pouches and straps can be sculpted nonfunctional.

### Hands

- Exactly four fingers per hand.
- Black fingerless glove shape.
- Visible saurian fingers and dark horn claws.
- Grip hand must hold route probe / flag marker / route-reader.
- Claws must be blunt and thick enough for printing.

### Feet / Boots

- Exactly four-toe / front claw read per foot.
- Field boot silhouette from KFR-REF-008.
- Broad stance platform.
- Toe bumper protects claws while preserving claw read.
- Tread blocks simplified into durable lugs.
- No real boot branding, logos, or product names.

### Tail

- Functional balance silhouette.
- Thick base and taper.
- Blunt tip.
- Simple paint zones: upper Route-Dust Olive Teal, underside Sage Green.
- Red proto-quill / ridge accents only if sculpted short, sparse, and sturdy.

## Accessory Compatibility

Use KFR-REF-011 for accessory scale and storage logic.

### Approved Prototype A Accessory Pool

1. GPR / route-reader device
2. black Path flag marker
3. route probe / collapsible staff
4. compact Mountaineer field pack
5. dark canvas EMT pouch, no symbol
6. route goggles
7. brass route token / whistle
8. carabiner / route clip
9. folded terrain map / route card
10. optional alternate gripping hand pair

### Grip / Storage Rules

- Route-reader should fit one gloved gripping hand and optionally clip to chest.
- Black Path flag marker should be blunt and grippable.
- Route probe / collapsible staff should fit tool grip hand.
- Field pack may be separate or sculpt-reference only; avoid thin removable straps on first prototype.
- EMT pouch may be sculpted onto harness or printed as separate pouch if size allows.
- Carabiner / token should be thick enough to print; tiny functional clips are not required.
- Route-card graphics are placeholder only.

Do not add generic survival weapons, ice axes, knives, fantasy blades, firearms, or ninja gear to Prototype A.

## Paint Integration Notes

Use the updated paint guide:

- `docs/toy-development/kavo-forest-rill-paint-guide.md`

Critical paint-engineering considerations:

- Keep checker trim in paintable, controlled zones.
- Keep badge symbols simple and readable, but not canonized.
- Use W-02 Light Route Dust lightly on boots, lower legs, tail contact, harness edges, and tools.
- Avoid tight paint rub at neck, shoulders, wrists, hips, and tail socket.
- Test primer + paint + clear coat before final joint tolerances are locked.

## Safety / Prototype Handling Notes

This is an adult collectible prototype.

Do not describe it as child-safe, kid-friendly, or toy-certified.

3D print safety:

- use ventilation
- use gloves and eye protection when handling resin
- avoid skin contact with uncured resin
- cure parts properly
- control sanding dust
- blunt sharp claws, quills, tail tip, flag marker, route probe, and boot claws
- keep brittle test parts away from children

## Recommended Immediate Production Sequence

1. Sculpt neutral blockout: head, torso, pelvis, limbs, boots, tail.
2. Test silhouette at 5-inch scale from front, side, and back.
3. Add primary head details from KFR-REF-005.
4. Add hand / glove / grip geometry from KFR-REF-007.
5. Add boot stance and toe clearance from KFR-REF-008.
6. Add tail socket and keyed removable tail from KFR-REF-009 / KFR-REF-012.
7. Add harness and pouches from KFR-REF-010.
8. Add accessory handle test pieces from KFR-REF-011.
9. Print joint test coupons.
10. Adjust tolerances before full prototype print.
11. Add final scale texture, badges, checker trim, and paint-mask relief.
12. Export first STL / 3MF part set for test printing.
