## Purpose

Use this checklist before detail sculpting begins.

The blockout stage should prove the figure works as a readable, printable, balanced 5-inch Kavo prototype before adding scales, stitch lines, badges, checker trim, tiny buttons, or other detail confetti.

This is the stage for big forms, silhouette, stance, garment layers, tail balance, watch scale, and joint socket planning.

## Blockout Pass Order

Recommended order:

1. scale / units setup
2. neutral figure massing
3. head / neck / snout silhouette
4. torso / pelvis massing
5. legs / boot stance
6. tail balance massing
7. arms / hands massing
8. garment layer massing
9. compact left-wrist watch massing
10. route harness massing
11. joint socket placeholders
12. accessory grip placeholders
13. shelf-read check
14. printability check
15. approval to enter detail sculpt

Do not detail early. Tiny details on bad proportions are expensive glitter on a crooked bridge.

## Core Silhouette Requirements

At thumbnail size, the blockout should read as:

- [ ] humanoid Velocisapien / Velociraptoroid, not human in reptile makeup
- [ ] saurian head with snout and brow
- [ ] chunky retro collector action figure proportions
- [ ] relaxed route-readiness, not mid-sprint
- [ ] strong tail silhouette with balance purpose
- [ ] broad field-boot stance
- [ ] route-worker jacket / harness upper silhouette
- [ ] dark cargo-pants lower silhouette
- [ ] compact left-wrist watch, not a wrist computer brick
- [ ] no ninja / assassin / monster / generic raptor scout read

## Head / Neck / Expression Blockout

- [ ] Saurian head shape blocked before shades detail.
- [ ] Snout length reads Velocisapien, not human nose.
- [ ] Brow mass supports sly, calm, dangerous expression.
- [ ] Mouth line position planned.
- [ ] Primary shades volume blocked as thick frame, not thin eyewear wires.
- [ ] Short sparse proto-quill mohawk volume placed.
- [ ] Proto-quills are short and thick enough for later print-safe sculpting.
- [ ] Neck joint volume planned without making neck too thin.
- [ ] Head can angle for route-reading side glance.

Blockout rejection signs:

- head reads mammal / human
- snout too tiny
- brow too soft
- quills too many, too long, or too needle-like
- shades hide all expression

---

## Torso / Pelvis Blockout

- [ ] Torso mass supports cropped jacket, field-knit under-layer, and harness.
- [ ] Chest is chunky enough for retro figure read.
- [ ] Waist is not too narrow for durability.
- [ ] Pelvis supports hip joints and tail socket.
- [ ] Belt / pants waistband can hide torso / pelvis seam if needed.
- [ ] Optional waist swivel seam tested visually but not required.
- [ ] Tail socket location planned before jacket / harness details.
- [ ] Torso does not drift into superhero armor or bare monster anatomy.

---

## Garment Layer Blockout: KFR-REF-014A

### Cropped Route Jacket

- [ ] Cropped jacket shell blocked over torso.
- [ ] Jacket length clears hips, tail base, and harness lower straps.
- [ ] Jacket collar volume blocked as sturdy shearling-style texture mass.
- [ ] Lapel / zipper line planned without thin fragile flaps.
- [ ] Sleeve and cuff volume clear shoulder and wrist movement.
- [ ] Back yoke / seam area reserved.
- [ ] Jacket reads route-worker cold-weather gear, not biker parody, superhero costume, or military uniform.

### Under-Layer

- [ ] Black field-knit / thermal under-layer mass included under jacket.
- [ ] Under-layer does not become a superhero shirt or logo canvas.
- [ ] Harness can sit over under-layer cleanly.

### Cargo Pants

- [ ] Dark route-worker cargo pants massed on legs.
- [ ] Thigh reinforcement panels blocked.
- [ ] Knee reinforcement panels blocked if visible.
- [ ] Cargo pocket volume placed without blocking hip swing.
- [ ] Optional utility pocket volume placed only if it does not crowd articulation.
- [ ] Pant cuffs taper into boot collar.
- [ ] Fabric folds planned as broad toy-readable planes, not thin wrinkle noise.
- [ ] No dangling cords, chains, or fragile straps.

---

## Tail Balance Blockout

- [ ] Tail base is thick and structurally believable.
- [ ] Tail emerges from correct pelvis / lower-back area.
- [ ] Tail clears jacket hem and harness lower strap zones.
- [ ] Tail tapers toward a blunt, safe tip.
- [ ] Tail can act as third balance point or near-contact support.
- [ ] Tail angle supports relaxed route-readiness.
- [ ] Tail does not read as a weapon spike.
- [ ] Tail key / socket volume reserved.
- [ ] Tail does not pull figure visually backward.

Balance blockout test:

- [ ] figure stands in neutral stance with tail installed
- [ ] boots carry most weight
- [ ] tail supports pose without becoming a kickstand gimmick
- [ ] field pack / accessory mass does not topple silhouette

---

## Legs / Boots Blockout

- [ ] Legs are stable and route-worker practical.
- [ ] Hip width supports balance.
- [ ] Optional knee hinge location is marked or deferred.
- [ ] Reinforced knee panel can hide optional knee seam if used.
- [ ] Boot collar meets pants cuff cleanly.
- [ ] Boots are broad enough for freestanding display.
- [ ] Four-toe claw clearance is reserved at front of each boot.
- [ ] Boot soles are flat enough for stance testing.
- [ ] No ankle articulation included for Prototype A.

---

## Arms / Hands / Watch Blockout

### Arms and Hands

- [ ] Shoulder joint volumes blocked.
- [ ] Arms clear jacket and harness bulk.
- [ ] Wrist swivel volumes blocked.
- [ ] Hands massed with exactly four fingers per hand.
- [ ] Black fingerless glove cuff volume planned.
- [ ] Grip hand volume can hold route probe / black flag marker.
- [ ] Claw volume kept blunt and print-safe.

### Compact Left-Wrist Watch: KFR-REF-014B

- [ ] Watch is on Forest's left wrist.
- [ ] Watch is compact at 5-inch scale.
- [ ] Watch reads as hard route-reader device, not wrist warmer.
- [ ] Watch case has rectangular field-device mass.
- [ ] Watch face / screen plane is visible but not oversized.
- [ ] Strap sits over or just above glove cuff / jacket cuff.
- [ ] Watch does not hide the black fingerless glove read.
- [ ] Watch does not block wrist swivel planning.
- [ ] Exposed wrist / hand areas remain Velocisapien scaled skin, never human skin.
- [ ] No real-world watch branding or exact trade dress is present.

---

## Harness / Gear Blockout

- [ ] Light route harness massed over jacket / under-layer.
- [ ] Harness does not block shoulder articulation.
- [ ] Chest badge zones placed but not detailed.
- [ ] Back harness panel placed but not overbuilt.
- [ ] Checker trim zones reserved as paint / relief areas only.
- [ ] EMT pouch location blocked with no symbol.
- [ ] D-ring / clip location reserved with print-safe size.
- [ ] Field pack location tested for balance if separate.
- [ ] Harness reads route gear, not tactical military uniform.

---

## Shelf-Read Check

Review the gray blockout from front, side, back, and three-quarter views.

- [ ] Front view reads Kavo / Forest immediately.
- [ ] Side view shows snout, chest, boot stance, and tail balance.
- [ ] Back view shows tail base, jacket back, harness back, and stance.
- [ ] Three-quarter view has relaxed menace and route-readiness.
- [ ] Watch is visible enough to matter but not so large it steals the figure.
- [ ] Tail supports attitude and balance.
- [ ] Figure feels precise, streetwise, and impossible to rush.
- [ ] Figure does not become lizard soup in tiny sunglasses.

---

## Printability Check Before Detail

- [ ] No planned part is too thin for resin prototype handling.
- [ ] Watch buttons / bezel / strap are chunky enough.
- [ ] Jacket cuffs and collar are not fragile.
- [ ] Cargo pockets are raised sturdy panels, not free-floating flaps.
- [ ] Proto-quills are short and sturdy.
- [ ] Claws are blunt and thick.
- [ ] Tail base and socket have enough material.
- [ ] Boot soles are flat and broad.
- [ ] Supports have likely hidden landing zones.
- [ ] First coupon exports can be derived from blockout geometry.
