# Kavo "Forest" Rill Parts Breakdown Spec

**Room:** Meridian Prime Toy Development Room / Action Figure Prototype Lab  
**Folder:** `production/toy-prototypes/kavo-forest-rill/sculpt-notes/`  
**Status:** First-pass parts, seams, sockets, and sculpt build order  
**Prototype Target:** 5-inch retro collector action figure  
**Canon Boundary:** Sandbox production file. Does not alter main canon.

---

## Purpose

This file defines how the first Kavo "Forest" Rill prototype should be separated into sculptable, printable, paintable, and assembleable parts.

The goal is to make a durable physical prototype, not a perfect digital statue with joints that explode like a tiny pressure boiler.

---

## Core Engineering Assumptions

**Prototype scale:** 5 inches tall, excluding tail reach.  
**Printing method:** Resin prototype, preferably tough or ABS-like resin for joints and tail.  
**Primary pose:** Relaxed route-readiness.  
**Articulation philosophy:** Simple retro articulation first; advanced articulation later.  
**Tail philosophy:** Essential balance and character feature, not decoration.

## Recommended First Build Configuration

Use this for the first physical test:

- head with sculpted shades
- torso with sculpted harness and badge cluster
- pelvis / hip block with tail socket
- left full arm with wrist socket
- right full arm with wrist socket
- two swappable hands or fixed gripping hands
- left leg with integrated foot
- right leg with integrated foot
- keyed removable tail
- route-reader device
- black Path flag marker
- route probe

Do not include elbow or knee joints in the first sculpt unless the figure is already stable and print tolerances are proven.

---

## Joint and Seam Strategy

### Neck Joint

Recommended:

- simple swivel peg or shallow ball joint
- peg rising from torso or socket inside head
- enough clearance for head turn without scraping shades or mohawk

Design notes:

- hide seam under jaw / neck folds
- keep neck thick enough for saurian strength
- avoid thin human-neck proportions

---

### Shoulder Joints

Recommended:

- classic swivel shoulders for first prototype
- cylindrical peg into torso socket
- shoulder seam hidden by jacket / harness / scale transition

Design notes:

- shoulders must clear route harness
- arms should rotate forward enough to hold route-reader or probe
- avoid tiny ball posts for first print

---

### Wrist Joints

Recommended:

- thick swivel peg hidden by claw-wraps
- optional fixed hands if wrist tolerance fails

Design notes:

- wrist wraps are seam camouflage
- gripping hands should be tested around accessory handles before final sculpt lock
- claws must not collide with handles

---

### Hip Joints

Recommended:

- simple T-joint or swivel hips
- strong pelvis block
- legs sculpted for wide stable stance

Design notes:

- do not sacrifice foot stance for articulation
- hip seam can hide under belt, harness straps, or clothing edge
- leg movement only needs enough range for relaxed stance variation

---

### Tail Joint

Recommended first version:

- keyed removable tail
- thick peg into pelvis / lower-back socket
- key shape prevents upside-down assembly
- round enough to evolve into swivel tail later if desired

Design notes:

- socket must not weaken pelvis wall
- tail base should look anatomical, not plugged-on
- tail should aid figure balance
- tail tip must be thick and safe

---

## Seam Placement Map

Preferred seam hiding areas:

- under jaw / neck folds
- shoulder clothing / scale transition
- wrist wraps
- belt line / lower harness
- tail base folds
- boot / ankle wrap transition if feet are separated later
- field pack contact area

Avoid seams across:

- face
- snout bridge
- brow expression
- shades / goggles lenses
- badge faces
- checker trim
- front center harness details
- visible top tail curve

---

## Sculpt Build Order

Recommended sculpt order:

1. Block full neutral body silhouette with tail attached as temporary mesh.
2. Confirm 5-inch scale and head-to-body proportion.
3. Block feet and tail balance before detailing anything.
4. Block head expression, snout, brow, and proto-quill mohawk.
5. Add hands with four-finger count and accessory grip cylinders.
6. Add route harness, badge cluster, wraps, and pouches.
7. Separate tail and create socket / keyed peg.
8. Separate arms, hands, legs, head, and pelvis according to joint plan.
9. Add broad scale texture zones.
10. Add checker trim and badge details at print-safe size.
11. Create accessories to match hand grip tests.
12. Export joint test coupons before exporting full figure.
13. Export full prototype parts.

## Accessory Handle Standards

To keep hands and accessories compatible, define one standard grip diameter before sculpt lock.

Recommended first grip standard:

- route probe handle: same grip diameter as black flag marker pole
- route-reader side grip: slightly larger, with flattened sides
- gripping hand: designed around a removable test cylinder

Checklist:

- [ ] one grip cylinder created
- [ ] gripping hand sculpted around cylinder
- [ ] route probe uses same grip standard
- [ ] flag pole uses same grip standard
- [ ] route-reader includes compatible side handle
- [ ] claws do not block insertion

## Accessory Part Breakdown

### Route-Reader Device

Parts:

- one-piece body preferred
- optional separate screen insert only for later deluxe print

Print notes:

- screen details can be painted
- knobs and buttons should be raised but not tiny crumbs
- handle must be thick

### Black Path Flag Marker

Parts:

- one-piece flag and pole preferred
- flag thickened for resin durability
- lower stake blunted

Print notes:

- print two or three copies for durability tests
- avoid ultra-thin flag edge

### Route Probe

Parts:

- one-piece probe preferred
- blunt tip
- visible grip zone

Print notes:

- slight taper is okay
- not a spear
- not needle-thin

---

## Printability Standards

First sculpt should follow these practical constraints:

- no required part should depend on a needle-thin feature
- quills should be short, sparse, and thick enough to survive handling
- claws should be expressive but blunted
- shades should be sculpted thick or treated as optional display parts
- badge details should be large enough to paint
- checker trim should be sculpted or paint-marked in broad readable zones
- sockets should have enough wall thickness
- pegs should be test printed before full figure

## Recommended Test Coupons

Create separate test files for:

1. neck peg / socket
2. shoulder peg / socket
3. wrist peg / socket
4. hip peg / socket
5. tail peg / socket
6. hand grip cylinder
7. route-reader grip
8. flag pole grip
9. route probe grip

Each test coupon should be labeled with clearance values.

Suggested resin clearance tests:

- 0.10 mm
- 0.15 mm
- 0.20 mm

Actual fit depends on printer, resin, exposure, cure time, and sanding.

---

## Paint and Assembly Considerations

Parts that should be paint-friendly:

- head separate from body
- hands separate from arms if wrist joints are used
- tail separate from pelvis
- accessories separate

Paint before final assembly when possible.

Watch for paint rub at:

- neck joint
- shoulder sockets
- wrists
- hips
- tail socket
- hand grips

Sealant should not fill socket tolerances.

## Durability Risk Register

| Risk | Severity | Mitigation |
|---|---:|---|
| Tail peg snaps | High | Thick keyed peg; tough resin; test coupon. |
| Figure falls backward | High | Wider feet; tail balance; adjust tail curve. |
| Shades break | Medium | Sculpt-on shades first; removable later. |
| Quills snap | Medium | Short sparse thick quills. |
| Flag pole snaps | Medium | Thickened pole; print multiples. |
| Wrist pegs fail | Medium | Use thicker pegs or fixed hands. |
| Badge details flake | Low | Overscale badges slightly. |
| Claws are sharp | Medium | Blunt and sand tips. |
