## Purpose

This file is the master index for the Kavo "Forest" Rill action figure prototype packet.

It ties together the canon-to-toy brief, figure spec, sculpt notes, print plans, test logs, paint guide, packaging concept, candidate canon log, and remaining action items for the first complete 3D-printable prototype workflow.

Use this index as the toy-lab map: canon on one shelf, sculpting on another, print gremlins in their labeled drawer, and paint decisions where they cannot wander off wearing goggles.

## Prototype Summary

**Character:** Kavo Rill / Forest Rill  
**Species:** Velocisapien / Velociraptoroid  
**Role:** Mountaineer / route guide / wilderness EMT / GPR path-reader  
**Toy Format:** 5-inch retro collector action figure prototype  
**Audience:** Adult collectible prototype, not child-safe, not safety-certified  
**Core Read:** Relaxed menace, precise route professional, streetwise, useful under pressure  
**Primary Prototype Goal:** A durable 3D-printable figure with expressive saurian anatomy, tail balance, route harness, shades / goggles, pressure-route badges, claw-wraps, and canon-based Mountaineer accessories.

## Required Canon Markers

The packet protects these markers across sculpt, print, paint, and packaging:

- Kavo Rill is his real name.
- Forest Rill is his human / crew name.
- Velocisapien / Velociraptoroid identity.
- Mountaineer / route guide / wilderness EMT / GPR path-reader role.
- Terrain truth function.
- mostly humanoid saurian body plan.
- expressive saurian face with snout and brow.
- scaled skin.
- universal tail.
- four-fingered clawed hands.
- four-toed clawed feet.
- short sparse red proto-quill mohawk.
- black-frame shades and / or route goggles.
- subtle checker trim.
- light route harness.
- pressure-route badge cluster.
- claw-wraps.
- relaxed posture that can become dangerous instantly.

## Guardrails

Do not make Kavo:

- a generic raptor scout
- a ninja
- an assassin
- an animalistic alien
- a dinosaur monster
- a human in reptile makeup
- a protected-toyline clone
- an over-armed fantasy combat figure
- a literal ska musician with stage props

Do not add hidden Book One revelations or unapproved canon through toy accessories, packaging stats, badge meanings, or medical symbols.

## Production Flow

Recommended sequence:

1. Review canon-to-toy brief.
2. Review figure spec and articulation plan.
3. Create / approve model sheet using prompt pack and turnaround checklist.
4. Create digital sculpt from first sculpt brief.
5. Build parts according to parts breakdown spec.
6. Export files according to STL planning notes.
7. Run joint tolerance test plan.
8. Run Batch A coupon / head / hand / accessory tests.
9. Run Batch B body / partial assembly / balance tests.
10. Run Batch C full raw prototype test.
11. Approve paint swatches.
12. Paint prototype using painted prototype plan.
13. Complete painted prototype review log.
14. Update this index with final status and remaining action items.
