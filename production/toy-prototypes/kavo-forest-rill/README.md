## Purpose

This folder collects production-facing files for the Kavo "Forest" Rill physical action figure prototype.

Use this area for sculpt references, print notes, paint tests, assembly notes, STL / 3MF planning, and prototype iteration logs.

The main canon repository remains read-only reference. All toy-development production notes belong here or under `docs/toy-development/` in this sandbox repository.

---

## Current Prototype Goal

Create a 3D-printable Kavo "Forest" Rill figure that preserves canon while translating him into a durable retro action figure format.

Target figure:

- 5-inch scale
- chunky but not goofy
- relaxed route-readiness pose
- expressive saurian face
- short sparse red proto-quill mohawk
- black-frame shades and / or route goggles
- light route harness
- pressure-route badges
- claw-wraps
- universal tail engineered for balance
- accessories based on Mountaineer / route-guide canon

## Prototype Safety Notes

This prototype is not for children.

When producing physical prints:

- use ventilation
- wear gloves with uncured resin
- avoid skin contact with resin
- cure all parts properly
- control sanding dust
- sand or blunt sharp points
- test brittle parts before handling
- avoid marketing as a child-safe toy

Commercial release requires qualified review of toy safety, age grading, small parts, sharp points, paint and material safety, labeling, and testing requirements.
