# Do Not Promote Log

> Sandbox log for useful ideas that should not leak into canon accidentally.

**Sandbox Status:** Workflow Log  
**Canon Status:** Not canon. Entries here are intentionally outside canon unless later re-reviewed.  
**Authoritative Promotion Rules:** `Discontent2/meridian-prime:docs/development/sandbox-promotion-rules.md`

## Purpose

This file preserves useful rejected, unsafe, contradictory, alternate-continuity, overexplaining, or adaptation-only ideas.

Do not treat this as a trash bin. Treat it as a quarantine shelf with warning labels.

## What Belongs Here

Use this log for ideas that are:

- interesting but canon-breaking
- too close to protected mysteries
- alternate-continuity only
- redundant with stronger canon
- useful for art, comics, toys, games, or prompts but not lore
- rejected after review
- too large for canon right now
- misleading if quoted without context

## What Does Not Belong Here

Do not use this file for:

- raw brainstorming that has not been reviewed
- normal sandbox experiments
- Candidate Canon material
- canon summaries
- protected mystery answers
- material that should simply be deleted because it is irrelevant or unsafe

## Core Rule

A Do Not Promote label does not mean an idea was worthless.

It means the idea is safer, stranger, cleaner, or more useful outside canon.

The sandbox should remain wild. Promotion should be disciplined.
