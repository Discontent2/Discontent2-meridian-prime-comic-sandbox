# Side Story Candidate Canon Log

**Sandbox Status:** Development Department Control Log  
**Label:** Candidate Canon Tracking  
**Canon Boundary:** Entries here are not canon. They are only candidates for later review in the main Meridian Prime canon/story process.

## Purpose

This log tracks sandbox inventions that may be worth reviewing for possible promotion into main canon.

Candidate Canon should be rare.

Do not mark an idea Candidate Canon just because it is interesting. Mark it only when it improves the larger Meridian Prime project, strengthens existing canon without rewriting it, and deserves deliberate review.

## Candidate Canon Process

1. Develop freely in the sandbox.
2. Identify a useful invention.
3. Add it to this log with reasoning.
4. Bring it to the main canon/story room for review.
5. Only after explicit approval may it be added to `Discontent2/meridian-prime`.

## Promotion Guardrail

Promoting a sandbox invention should answer:

1. Does it strengthen existing canon?
2. Does it avoid solving protected mysteries?
3. Does it create future story utility?
4. Does it fit the tone, terminology, and world logic already established?
5. Is it worth bringing into the canon library rather than leaving as sandbox texture?

If the answer to any of these is weak, leave it in the sandbox.
