# Canon Borrowing Map

**Sandbox Status:** Development Department / Source Notes  
**Label:** Canon Reference + Development Concept  
**Canon Boundary:** This file maps what sandbox work may borrow from main canon. It does not promote sandbox inventions into canon.

## Source-of-Truth Priority

When developing side stories, use the main repo as reference in this order:

1. `docs/traverse-system.md`
2. `docs/species-guide.md`
3. `docs/visual-development/visual-style-guide.md`
4. `docs/story/book-one-novella-structure.md`
5. `docs/story/book-one-novella-beat-grid.md`
6. `docs/characters/tenet-nocsid-iii.md`
7. `docs/legacy/the-last-traverse.md`
8. Chapter planning and draft files when specific tone or prose texture is needed.

- On Traverse culture
- Prime Ops procedures

- The Trail / The Path distinction
- black flags
- Named Cracks
- Growth Events
- crystal storms
- Circle Camp / Storm Hold

- Winter-Over equipment culture

### Visual Language

Safe to borrow:

- cold industrial survival
- patched machinery
- procedural danger
- official control vs field truth
- beauty under lethal conditions
- cold steel, frost, vapor, field lights, cab interiors
- official clean paperwork contrasted against grease-pencil field truth

### Species and Cultures

Safe to borrow:

- Saurasapien / Aeonolacertian morph-type logic
- pressure-rhythm aesthetics
- Antisapien / Absconditian signal-band logic
- bio-cabling
- Meridian class-culture visual lanes
- Dieselwright, Scarwalker, Ghostwaver, Winter-Over, and Prime Ops workplace cultures

### Story Atmosphere

Safe to borrow:

- paperwork can lie
- routes remember
- old infrastructure answers
- machines reveal truth under stress
- field crews know what official records soften
- local myths may be useful without being literally true

## Borrow With Caution

Use lightly and do not define further:

- Aeonos / Àæonos distinction
- Core-adjacent rumors
- gateway-adjacent imagery
- old route records
- Horus's authority footprint
- NCI as background institution
- Rob's public legacy
- The House Ten Built as rumor only

## Do Not Borrow as Plot Engine Without Review

Avoid using these as the central mechanism of a sandbox story unless specifically approved:

- Tenet Nocsid III
- MITE II
- Red Umbrielor
- Rob's poisoning
- the assassin inside NCI
- Horus's full motive
- the hidden formation near the Claw
- Grandfather Ten's pistol
- The House Ten Built
- full Core / gateway truth

## Borrowing Rule of Thumb

If a canon ingredient explains how the world works, borrow it.

If a canon ingredient explains why the main saga matters, protect it.
