# Toy Candidate Canon Log

**Room:** Meridian Prime Toy Development Room / Action Figure Prototype Lab  
**Repository:** Discontent2/Discontent2-meridian-prime-comic-sandbox  
**Status:** Sandbox review log  
**Canon Boundary:** Nothing in this file becomes main canon unless separately reviewed and approved in the main Meridian Prime canon process.

---

## Purpose

This log catches useful toy-development ideas that might deserve later Meridian Prime canon review.

Toy development often invents practical shapes, symbols, colors, tool names, packaging phrases, badge icons, and accessory functions. Some may improve the larger project. Others are only useful for plastic goblin engineering. This document keeps them sorted before they sneak into canon wearing tiny boots.

---

## Label Definitions

### Canon Reference

Directly drawn from the main repo. Must be preserved.

### Toy Adaptation Choice

A toy-specific simplification or exaggeration for sculptability, printability, articulation, durability, or shelf presence.

### Prototype Engineering Choice

A practical decision for 3D printing, joint strength, part separation, supports, assembly, or paint.

### Toy-Only Continuity

A non-canon toy-line embellishment.

### Candidate Canon

A toy-development idea that might improve the larger Meridian Prime project and should be reviewed separately.

### Rejected / Do Not Use

A concept that harms canon, printability, safety, or originality.

## Review Process

When a toy-development idea seems valuable beyond the toy room:

1. Add it here.
2. Label it Candidate Canon.
3. Explain why it helps.
4. Explain what it might break.
5. Do not update main canon until separately approved.

This file is a quarantine jar for shiny ideas. Shake gently.
