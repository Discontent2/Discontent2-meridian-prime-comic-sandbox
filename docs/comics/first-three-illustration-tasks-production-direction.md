## Global Art Direction

Use the following shared art rules for the first three illustration tasks.

```text
Style: hybrid comic realism
Line weight: thin-to-thick line weight for depth
Color finish: cel-shaded with grit, weathering, dust, frost, and industrial wear
Sheet format: production model sheets
Label style: labels outside the art or in a clean side legend
```

Avoid crowding drawings with excessive embedded labels. Essential in-world markings may appear directly on objects, such as vehicle call numbers, faded stencils, decals, and packet text.

## Task 1: Red Umbrielor Master Model Sheet

### Core Identity

```text
Red Umbrielor
MMS Quadtrack 580
Call number 289
```

Use this naming stack for comic production.

**Important:** `289` is Red Umbrielor's call number, not the tractor model.

### Required Sheet Views

The Red Umbrielor master model sheet should include:

- full exterior side view
- front three-quarter view
- rear three-quarter view showing hitch and coupling hardware
- blade detail inset
- cab detail inset
- weathering detail strip
- one cold-start angle

### Exterior Design Requirements

Red Umbrielor must read as a massive red articulated quad-track Traverse tractor built for crystalline-regolith route work.

Required visual anchors:

- oxidized industrial red body
- mismatched patched red panels
- scarred, mineral-dusted, frost-cracked, heat-cycled surface wear
- permanently mounted front blade
- slightly boxy multidirectional route-shaping blade
- replaceable serrated blade edge
- heavy rear hitch and reinforced coupling hardware
- saddle tanks above rear tracks
- `Red Umbrielor` stenciled on saddle tanks
- `Red Umbrielor` stencil is faded and partly unreadable in places
- `289` stenciled on the side of the engine compartment
- `289` should feel practical, not decorative

Avoid:

- sleek sci-fi vehicle language
- military tank silhouette
- clean showroom finish
- polar snow-vehicle identity
- optional-looking blade
- decorative machine greebles with no mechanical logic

### Cab Inset

The cab inset should be technical first.

Include:

- analog/digital hybrid controls
- MD-8 gauge
- turkey roaster / Waviston heater in safety cage
- under-seat compartment location
- passenger seat may be visible, but it should not dominate

The cab should feel like a lived-in work environment, not a fighter cockpit or luxury vehicle.

### Cold-Start Angle

The cold-start angle should emphasize:

```text
work realism + emotional atmosphere
```

Scene guidance:

- NCI staging yard before dawn
- frost, vapor, cold lights, scarred red steel
- machine waking under field procedure
- optional tiny human scale silhouette
- beloved family machine first
- industrial beast by sheer mass second

Do not stage Red Umbrielor like a superhero vehicle.

## Task 2: Main Cast Model Sheet Lineup

### Sheet Format

Each character should include:

- front view
- side view
- three small expression notes
- silhouette read
- outfit notes
- approximate height / scale comparison

Approximate height tiers are enough. Exact height numbers are not required for this first production pass.

## Task 3: MITE II Convoy Sheet

### Sheet Format

The MITE II convoy sheet should include:

- main travel-mode side / three-quarter hybrid diagram
- simplified Traverse Town State A view
- State A shown from a standing three-quarter viewpoint
- clean side legend instead of labels directly on the art
- small top-down inset for layout clarity

---

### Travel Mode Order

#### 409

409 is the lead sensor tractor.

Required details:

- red NCI-modded crystal groomer
- pulls nothing
- `409` marked on cab doors
- front blade
- 20-foot blade-mounted GPR boom
- 8-foot mast with pulley
- electric winch cable supporting boom tip
- rugged GPR unit at leading end
- Ildi drives
- Forest rides in rear terrain-reading station

409 should feel compact, sensor-forward, practical, and colder / less sentimental than Red Umbrielor.

---

#### Red Umbrielor Chain

```text
Red Umbrielor / MMS Quadtrack 580 / call 289
→ Comms/Kitchen Mod
→ Solar/Berthing Mod
→ Tool
→ two Magic Carpets
→ four black MD-8 bladders
```

Red Umbrielor remains the command / life-support tractor and the central visual anchor of MITE II.

#### 287 Discovery Chain

```text
287 Discovery
→ Refrigerator/Storage Mod
→ Shed
→ Tool
→ two Magic Carpets
→ four black MD-8 bladders
```

### 287 Discovery Visual Rules

Required details:

- used-machine base color: faded yellow/orange
- red NCI additions
- red knuckle boom
- front winch
- `DISCOVERY` decal visible across windshield
- no visible `Tight Tracks` marking

`Tight Tracks` remains verbal only.

---

### Mods

MITE II Mods should read as:

- red modified double-wide 45-foot high-cube containers
- dust-muted industrial Mod red
- platform-mounted
- four unpowered straight tracks
- boxy
- practical
- repaired
- insulated
- field-modified
- heavy enough to matter
- ugly enough to trust

Avoid:

- sleek sci-fi pods
- luxury mobile-home design
- powered Mod tracks
- hi-track snow module logic
- military armored trailer identity

---

### Fuel Train

Fuel train structure:

```text
Tool
→ two Magic Carpets side by side
→ four smooth black rubber MD-8 bladders
```

Fuel train rules:

- Tool is a towable spreader bar
- two Magic Carpets ride side by side
- four black MD-8 bladders
- bladders are smooth black rubber
- black patches only
- no warning labels
- no big bladder numbers
- hose connector numbers only if visible in detail inset

---

### Traverse Town State A

Traverse Town State A should be shown from a standing three-quarter viewpoint.

Required State A features:

- tractors still attached at heads of their train rows
- two long parallel Mod rows
- tractor-width central alley
- fuel tails still attached
- 409 parked forward / near service side with GPR boom still attached
- Aquifer Drill visible as folded / stowed detail on Comms/Kitchen Mod
- no freestanding street lights
- no Traverse Town sign
- Mod-mounted lights only

State A should feel like a convoy that has just parked into a working town shape, not a permanent base.

Use this order for the first three illustration tasks:
