## Core Production Principle

MITE II is not one tractor pulling a vague row of boxes.

MITE II is a small working Traverse convoy with three distinct machine roles:

1. **409** = lead sensor / terrain-reading tractor.
2. **289 Red Umbrielor** = command / life-support tractor.
3. **287 Discovery** = repair / storage / sustainment tractor.

Each tractor has a job, silhouette, crew association, and load logic.

The convoy should feel like:

> a moving industrial survival system built from old iron, field modifications, black fuel, practical grudges, and people who know which sounds mean death.

## 409: NCI-Modded Crystal Groomer

**Adaptation Label:** Candidate Canon / Sandbox Production Decision

### Core Identity

409 is a compact lead tractor based on the logic of a ski-resort snow shaper / groomer, but fictionalized as an **NCI-modded crystal groomer** for Meridian Prime.

It is not a snow vehicle in-world.

It is a crystalline-regolith route-reading platform.

### Naming and Numbering

```text
409 = tractor number / call number
No name
No nickname
```

409 should remain nameless.

This makes it feel more functional and less sentimental than Red Umbrielor or Rudy's used tractor.

### Crew Association

- **Ildi Vey** drives 409.
- **Kavo "Forest" Rill** rides in the rear cab and reads terrain while the Traverse is in motion.
- Forest always reads terrain when the Traverse is moving.
- 409 pulls nothing.

### Visual Role

409 is the forward sensor machine.

It should look:

- compact
- tracked
- low and controlled
- practical
- sensor-forward
- modified by NCI
- built to place instruments ahead of the convoy
- distinct from Red Umbrielor and Rudy's tractor

### Blade and GPR Boom System

409 has a front blade.

The front blade mounts the GPR boom system:

- **20-foot boom** mounted to the blade.
- **GPR unit** mounted at the leading end of the boom.
- Leading end of boom is supported by an **electric winch cable**.
- Cable runs through a **pulley** on an **8-foot mast** mounted on the blade.

The system should feel field-engineered but competent.

### Visual Design Notes

The GPR boom is the key silhouette feature.

Artist should show:

- front blade as structural mount
- 8-foot mast rising from or mounted to blade hardware
- pulley at mast top
- electric winch cable supporting boom tip
- 20-foot boom extending forward
- GPR unit at leading end
- reinforced pivot or hinge near blade
- protected cable routing
- dust and crystal abrasion on lower hardware

Avoid:

- crane boom that looks like cargo handling
- sleek sci-fi scanner arm
- magical sensor wand
- high-track snow vehicle read
- 409 towing cargo

### Production Function

409 gives the comic a visible way to show terrain intelligence before Red Umbrielor commits its mass.

It makes Ildi and Forest active in the machinery language:

- Ildi drives and monitors systems / signal behavior.
- Forest reads terrain from the rear cab / readout.

## 289 Red Umbrielor

**Adaptation Label:** Candidate Canon / Sandbox Production Decision

### Core Identity

Red Umbrielor is Tenet's signature Traverse machine and the comic's main vehicle anchor.

Corrected sandbox identity:

```text
MMS Quadtrack 580 = tractor size / class
Red Umbrielor = vehicle name
289 = call number
```

### Main Canon Loose Bolt

The main canon repo currently uses `MMS Quadtrack 289, "Red Umbrielor"` language. The sandbox correction treats **580** as the tractor class/size and **289** as the call number.

This should be treated as **Candidate Canon** until main canon is deliberately patched.

### Model and Class Notes

- There are smaller MMS Quadtrack models.
- Smaller models are retired from Traverse use.
- The active Traverse version for Red Umbrielor is the **MMS Quadtrack 580**.
- Red Umbrielor should look almost identical in core silhouette to a heavy articulated four-track tractor in the Case Quadtrac 580 family, but adapted for Meridian Prime.

### Marking Rules

Red Umbrielor's markings are specific:

- **289** is stenciled only on the side of the engine compartment.
- **Red Umbrielor** is stenciled above the rear tracks on the side of the saddle tanks.

Production meaning:

- `289` = practical call number.
- `Red Umbrielor` = machine identity / personality mark.

Do not scatter the number everywhere unless a later design pass approves additional small maintenance markings.

### Blade

Red Umbrielor always has a front blade.

Blade type:

- multidirectional
- open, slightly box-style blade
- replaceable serrated blade edge

The blade should feel like serious route equipment, not a cartoon dozer blade.

Use it to imply:

- route shaping
- crust breaking
- regolith pushing
- pressure-ridge testing
- growth-roller negotiation
- obstruction clearing

### Fuel and Tanks

Red Umbrielor has:

- saddle tanks above the rear tracks
- MD-8 fuel gauge

The saddle tanks also carry the Red Umbrielor name stencil above the rear tracks.

### Cab Layout

Red Umbrielor has a work-cab, not just a basic tractor cockpit.

Cab includes:

- main operator seat
- passenger seat / work seat
- extra jump seat or fold-down station
- under-seat compartment below the seat
- hybrid analog / digital controls
- MD-8 fuel gauge
- practical field controls
- sightlines inspired by a heavy Quadtrac-style cab

### Under-Seat Compartment

Grandfather Ten's pistol is stored below the seat.

Current note:

- Location is below the seat.
- Whether the compartment is factory-built, NCI-modified, or Ten/Rob-modified remains open.

### Turkey Roaster / Waviston Heater

The turkey roaster is now defined as an **NCI-modified Waviston auxiliary cab heater**.

Reference size:

- approximately 8 inches wide
- approximately 1 foot tall
- approximately 6 inches deep

Placement:

- left-front side of the cab
- left of steering and instrument cluster
- inside a safety cage

Visual notes:

- should look like a practical auxiliary heater
- should feel field-modified rather than glossy
- should be small but memorable
- should not be branded with a real-world manufacturer

Production language:

> NCI-modified Waviston cab heater.

Crew slang:

> turkey roaster.

### Rob Memory Cue

Rob's passenger-seat absence may be evoked by:

- cheap cologne lingering in the cab

This is a strong sensory cue for grief, memory, and Rob's presence-in-absence.

### Red Umbrielor Pull Chain

Red Umbrielor pulls the life-support / command chain:

```text
Red Umbrielor
→ Comms Mod
→ Solar/Berthing Mod
→ Tool with two Magic Carpets carrying four black fuel bladders total
```

Red Umbrielor does not pull the entire MITE II convoy by itself.

## 287 Discovery / Rudy's Tractor

**Adaptation Label:** Candidate Canon / Sandbox Production Decision

### Core Identity

Rudy drives a used MT865-style crawler tractor.

Current structure:

```text
287 = tractor number / call number
Discovery = old windshield decal from original owner
Tight Tracks = verbal crew nickname only
```

### Discovery Decal

Rudy's tractor was bought used.

The original owner had decaled **DISCOVERY** across the top of the windshield.

NCI kept the decal or never bothered to remove it.

This creates useful visual archaeology:

- Discovery is not necessarily an official NCI name.
- Discovery is not the current crew nickname.
- Discovery is old owner residue with accidental meaning.

### Tight Tracks Nickname

Rudy's tractor is verbally nicknamed:

```text
Tight Tracks
```

Rules:

- no visible `Tight Tracks` stencil
- nickname is verbal only
- no painted nickname on machine

Production meaning:

- `287` = current field identifier
- `DISCOVERY` = inherited windshield decal
- `Tight Tracks` = crew speech

### Visual Reference

Real-world reference type:

- MT865-style crawler tractor

In-world production description:

> used NCI-modified MT-class crawler tractor with front recovery winch and rear knuckle-boom crane.

### Equipment

Rudy's tractor has:

- front winch
- rear Fassi-style crane reference / in-world knuckle-boom crane

Use real crane logic as reference only. Do not label the crane with real-world brand names in finished art unless explicitly approved.

### Crew Association

Rudy Calder is the repair / sustainment authority.

His tractor is the repair spine of MITE II.

### Rudy Pull Chain

Rudy's tractor pulls:

```text
287 Discovery
→ Refrigerator/Storage Mod
→ Shed
→ Tool with two Magic Carpets carrying four black fuel bladders total
```

### Visual Role

287 should look:

- used
- modified
- mechanically trusted
- recovery-ready
- denser with repair hardware than Red Umbrielor
- visually tied to Rudy's machine truth role

Avoid:

- clean new machine look
- military recovery vehicle glamour
- visible Tight Tracks stencil
- over-designed sci-fi crane

### Core Mod Body

MITE II Mods are modified conex / seavan-style containers.

They ride on a platform attached to:

- four unpowered straight tracks

Important wording:

- straight tracks
- not hi-tracks
- unpowered
- platform-mounted
- pulled by tractors

### Mod Visual Grammar

All Mods should share the same base design grammar:

```text
modified conex/seavan body
on a platform with four unpowered straight tracks
pulled by tractor
not self-powered
```

### Exterior Design Logic

Mods should look:

- boxy
- practical
- repaired
- insulated
- field-modified
- function-stenciled
- heavy enough to matter
- ugly enough to trust
- adapted to crystalline regolith

They should not look like:

- sleek sci-fi pods
- polished mobile homes
- smooth aerospace modules
- luxury trailers
- pure farm wagons

### Shared Mod Features

Possible shared features:

- tow tongues

- cable lines
- vents
- filters
- ladder rungs

- patched insulation
- heat/cold cycle damage

## Fuel System and MD-8

**Adaptation Label:** Candidate Canon / Creator Decision for Main Canon Patch

### MD-8 Fuel

Creator decision from Q&A:

```text
MD-8 shall be made canon.
```

Use `MD-8` as the official fuel term going forward in comic production unless main canon later revises it.

No separate slang term is needed yet. Crew may simply say MD-8.

### Red Umbrielor Fuel UI

Red Umbrielor has:

- MD-8 fuel gauge
- hybrid analog / digital cab systems

### Fuel Handling Hardware

Field fueling hardware:

- pump and hoses lashed to a small sled
- sled and pump live on the Magic Carpet when not in use
- no warning labels

### Fuel Bladder Rules

Fuel bladders are:

- always black
- black patches only
- numbered at the hose connector
- numbers engraved on hose connector
- no large visible bladder numbers required in normal panels

### No Warning Labels

Creator decision:

```text
No warning labels.
```

Danger is communicated by procedure, black hardware, fuel handling behavior, and crew discipline rather than label clutter.

## Tool / Magic Carpet / Fuel Train Layout

**Adaptation Label:** Candidate Canon / Sandbox Production Decision

### Tool Language

Tools are known simply as:

```text
Tools
```

Common speech:

- "Go get a Tool."
- "Parts came in for Tool 12."

### Tool Numbering

Tool numbers:

- are not visibly displayed in normal panels
- exist as small engraving on the tongue

### Tool Function

A Tool is a towable spreader bar used to attach two Magic Carpets side by side.

### Magic Carpet / Bladder Arrangement

Four bladders ride in pairs of two.

Layout:

```text
Tool / spreader bar
├─ Magic Carpet A
│  ├─ black fuel bladder 1
│  └─ black fuel bladder 2
└─ Magic Carpet B
   ├─ black fuel bladder 3
   └─ black fuel bladder 4
```

Bladder placement:

- two bladders front-to-back on one Magic Carpet
- another two bladders front-to-back on another Magic Carpet
- the two Magic Carpets ride side by side
- both Magic Carpets attach to the Tool / spreader bar

This layout applies to both:

- Red Umbrielor's fuel Tool train
- Rudy's fuel Tool train

### Production Notes

The fuel train should feel:

- black
- heavy
- procedural
- low-slung
- ugly-useful
- drag-friendly
- dangerous without shouting about danger

Do not over-label.
