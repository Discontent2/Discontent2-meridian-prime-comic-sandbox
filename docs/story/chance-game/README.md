# Meridian Prime Chance Engine

**Status:** Sandbox infrastructure / Non-Canon

This folder stores reusable material generated through Meridian Prime d20 chance rooms.

Core principle:

```text
NPC Generator = casting bench
Location Generator = set-design bench
Story Generator = plot engine
Sandbox repo = memory palace
Canon repo = protected archive
```

All generated NPCs, locations, story seeds, drafts, and roll logs remain sandbox material unless explicitly reviewed and promoted.

Do not treat these entries as main canon facts.

## Folders

- `roll-logs/` stores roll receipts and session records.
- `story-seeds/` stores generated chance story premises and outlines.
- `drafts/` stores non-canon prose, comic, tabletop, or game drafts.
- `candidate-canon-review/` stores review notes for items being considered for promotion.
- `npc-pool/` stores reusable sandbox NPC cards.
- `location-pool/` stores reusable sandbox location cards.

## Canon Safety

Generated material may create pressure around protected mysteries, but may not solve them. A roll can point toward a protected mystery. It cannot answer it.
