# Ecology Name Generator

Classification: Sandbox ecology development only. Non-Canon. Requires Canon Review Before Promotion.

## Naming Principle

Habitat shapes the body. Behavior shapes the field name. Local fear shapes the warning name. Faction use shapes the registry name. Story pressure decides which name lies.

Generated names are sandbox generated, style-valid, and not finalized taxonomy canon.

## Name Layers

Each ecology seed may include:

- Field Name
- Local Name
- Registry Name
- Hunter / Forager Name
- Route-Crew Name
- Black-Market Name
- Sacred / Taboo Name
- Scientific Placeholder Name
- Known As

## Apex Predator Naming Logic

Locals name the predator by what it costs them. Officials name it by what they hope to control.

Example active seed: The Third Reflection.
