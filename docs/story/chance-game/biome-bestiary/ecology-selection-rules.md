# Ecology Selection Rules

Classification: Sandbox ecology development only. Non-Canon. Requires Canon Review Before Promotion.

## Core Principle

Ecology is infrastructure that eats, blooms, moves, rots, remembers, or lies.

A creature is not a monster first. A creature is a pressure system with teeth, wings, roots, spores, hunger, or memory.

## Use Scale

- Local organism: safe for sandbox stories if contained.
- Regional species: requires route and seasonal review.
- Biome / habitat system: requires topology and ecology review.
- Planetary ecological law: requires full canon review.
- Protected mystery ecology: hint only, do not solve.

## Apex Predator Bias Mode

Use Apex Predator Bias Mode when generating apex predators, top predators, major predators, dangerous wildlife, large carnivores, route predators, ambush predators, megafauna, territorial beasts, hunting animals, stalking animals, wet-season predators, dry-season predators, storm predators, Craton predators, Hydropolis predators, Eschar predators, Blackstep predators, or Mirror Basin predators.

Apex principle: An apex predator is not just a monster. An apex predator is a living jurisdiction.

Core guardrail: A predator may rule a place. It does not automatically rule the planet.
