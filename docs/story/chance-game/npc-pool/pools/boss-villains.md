# Boss Villains NPC Pool

**Pool Status:** Sandbox NPC Pool  
**Canon Status:** Non-Canon  
**Review Status:** Requires Canon Review Before Promotion  

This pool stores high-risk boss-villain NPC seeds for Meridian Prime chance stories, SNES-style adaptation concepts, comic encounters, tabletop boss fights, and late-game antagonist concepts.

Default contents are:

- Sandbox NPCs
- Non-Canon
- Requires Canon Review Before Promotion
- Usually Tier 4 Restricted / Use Carefully

## Pool Scope

Use this pool for NPCs such as:

- WWISSaRD Released Assets
- end-stage bosses
- hidden bosses
- secret bosses
- SNES adaptation bosses
- off-books stasis assets
- bio-hacked boss characters
- high-risk one-shot villains
- late-game antagonist concepts
- bosses with phase structures, arenas, and non-combat weaknesses

## Required Guardrails

Boss villains in this pool must not casually:

- solve the Core
- define Dry Port truth
- confirm gateway mechanics
- reveal Rob Holliday’s full death-chain
- identify the NCI assassin
- reveal Ildi’s restricted truth
- overwrite Book One
- canonize a secret WWISSaRD boss program without review
- provide real-world biotech, medical, genetic, surgical, chemical, extraction, weapon, or harm instructions

Core safety rule:

> The boss may be released near a mystery. The boss does not explain the mystery.
