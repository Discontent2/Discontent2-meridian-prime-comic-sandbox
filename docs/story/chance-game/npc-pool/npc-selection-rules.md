# NPC Selection Rules

**Status:** Sandbox infrastructure / Non-Canon

Use this file when the Chance Story Game Room needs to pull an existing NPC from the pool.

## Selection Order

1. Match Story Use.
2. Match Location.
3. Match Faction.
4. Match Species / People, if relevant.
5. Match Tone.
6. Check Availability.
7. Check Spoiler Visibility.
8. Check Canon Risk.
9. Check Pull Weight.
10. Check Do Not Use For.
11. Pull existing NPC or generate a new one.

## Pull Guidance

- Prefer Tier 1 and Tier 2 NPCs for ordinary story use.
- Use Tier 4 NPCs only when the scene actually needs their restricted pressure.
- Do not use any NPC to solve protected mysteries.
- Do not let an NPC replace a main cast function.
- Use primary pool for story pressure and secondary tags for retrieval.

## Response Format for Pull Requests

When asked to pull an NPC, return:

```text
Best match
Backup option
Reason for fit
Canon safety note
How to introduce them
How they help
How they complicate
Voice sample
```

## Final Principle

An NPC is useful before they are important.
