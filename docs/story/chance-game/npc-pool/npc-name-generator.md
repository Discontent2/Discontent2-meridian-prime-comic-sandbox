# NPC Name Generator Notes

**Status:** Sandbox infrastructure / Non-Canon

Generated names are sandbox generated, style-valid, and not conlang canon.

Core principle:

```text
Species shapes the sound.
Faction shapes the paperwork.
Crew shapes the nickname.
Story pressure decides which name hurts.
```

## Name Layers

Use as needed:

- Personal Name
- Family / clan / community name
- Crew name
- Registry name
- Faction title
- False name
- Known As

## Name-Conflict Logic

If an NPC has bad paperwork, misclassification, faction conflict, registry trouble, hidden identity, protected status, exile history, or a false official record, consider giving them multiple name layers.

A wrong name can be a wound, a weapon, or a locked door.

## Status Language

Every generated NPC card should include:

```text
Name Status: Sandbox generated / style-valid / not conlang canon
```

## Duplicate Name Check

Before saving a new NPC, check `npc-master-index.md` for name collisions. Repeated names are allowed only when intentional.
