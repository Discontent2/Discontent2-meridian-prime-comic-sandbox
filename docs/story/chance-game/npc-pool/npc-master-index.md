## Index Maintenance Notes

When adding a new NPC:

1. Check this index for name collisions.
2. Choose the primary pool based on story pressure, not paperwork.
3. Add only a lightweight row here.
4. Keep the full card in the appropriate pool file.
5. Preserve the sandbox / non-canon status unless the user explicitly approves canon review and promotion.
