# Meridian Prime NPC Pool

**Repository Status:** Sandbox / development only  
**Canon Status:** All entries are Non-Canon unless explicitly promoted after canon review.  
**Promotion Rule:** Requires user approval and canon review before any NPC may be treated as canon.

This folder stores sandbox NPCs for Meridian Prime chance stories, side-story seeds, comic one-shots, tabletop quests, game missions, anthology concepts, and prose drafts.

Core principle:

> The Story Game creates plot pressure. The NPC Pool creates people for the pressure to hit.

## Canon Safety

NPCs may reference canon pressure, but they may not alter canon or solve protected mysteries.

A roll can point toward a protected mystery. It cannot answer it.

## Pool Placement Principle

An NPC’s primary pool is where their story pressure lives, not merely who signs their paperwork.

## Name Status

Generated names are sandbox generated, style-valid, not conlang canon, and require review before canon promotion.
