# Meridian Prime Location Pool

**Repository Status:** Sandbox / development only  
**Canon Status:** All entries are Non-Canon unless explicitly promoted after canon review.  
**Promotion Rule:** Requires user approval and canon review before any location may be treated as canon.

This folder stores sandbox location seeds for Meridian Prime chance stories, side-story seeds, comic one-shots, tabletop quests, game missions, anthology concepts, visual development, and prose drafts.

Core principle:

> The Story Game creates plot pressure. The Location Pool creates places where the pressure becomes visible.

## Purpose

The Location Pool is a reusable set-design bench for the Meridian Prime Chance Engine.

Use it to:

- pull existing locations into chance stories;
- prevent every story from inventing disconnected set pieces;
- preserve roll receipts, canon risk, route impact, and seasonal logic;
- track which locations are safe, recurring, restricted, or retired.

All generated locations remain:

```text
Sandbox Location Seed
Non-Canon
Requires Canon Review Before Promotion
```

## How the Story Game Room Should Use This Pool

1. Start with `location-master-index.md` for fast set selection.
2. Choose a location whose Story Use, Region, Jurisdiction, Season, Visual Hook, Pull Weight, and Canon Risk fit the scene.
3. Open the relevant pool file for the full location card.
4. Check `Do Not Use For` before using the location.
5. Generate a new location only when no existing one fits.

## Saving Entries

Save each location into the pool where its story pressure lives, not merely where an official map would file it.

Where available, preserve roll receipts in the location card. Do not invent missing receipts.

## Canon Safety

Locations may reference canon pressure, but they may not alter canon or solve protected mysteries.

A roll can point toward a protected mystery. It cannot answer it.

Do not use locations from this pool to solve the Core, define Dry Port truth, reveal Ildi’s restricted truth, reveal Rob Holliday’s full death-chain, identify the NCI assassin, overwrite Book One, or redraw planetary route topology.

## Promotion to Canon

Candidate Canon requires explicit user approval and canon review. Do not add sandbox locations to main canon indexes unless promotion is approved.

## Name Status

Generated names are sandbox generated, style-valid, not canonized geography, and require review before canon promotion.

## Final Principle

The pool organizes locations by story usefulness first, lore importance second.
