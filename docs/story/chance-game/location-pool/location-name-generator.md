# Location Name Generator Notes

**Status:** Sandbox infrastructure / Non-Canon

Generated names are sandbox generated, style-valid, and not canonized geography.

Core principle:

```text
Geography shapes the image.
Jurisdiction shapes the official name.
Workers shape the field name.
Story pressure decides which name lies.
```

## Name Layers

Use as needed:

- Official Name
- Field Name
- Local Name
- Black-Market Name
- Sacred / Ceremonial Name
- Registry Name
- False Map Name
- Known As

## Name-Conflict Logic

If a location has contested jurisdiction, black-market use, bad maps, false records, protected users, or seasonal access shifts, consider giving it multiple name layers.

A wrong place-name can be a warning, a lie, a joke, a receipt, or a trap.

## Status Language

Every generated location card should include:

```text
Name Status: Sandbox generated / style-valid / not canonized geography
```

## Duplicate Name Check

Before saving a new location, check `location-master-index.md` for name collisions. Repeated names are allowed only when intentional.
