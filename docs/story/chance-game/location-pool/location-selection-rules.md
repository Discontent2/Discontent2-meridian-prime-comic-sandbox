# Location Selection Rules

**Status:** Sandbox infrastructure / Non-Canon

Use this file when the Chance Story Game Room needs to pull an existing location from the pool.

## Selection Order

1. Match Story Use.
2. Match Region / Route System.
3. Match Jurisdiction.
4. Match Season.
5. Match Tone / Visual Motif.
6. Check Canon Risk.
7. Check Route / Jurisdiction Impact.
8. Check Seasonal Impact.
9. Check Pull Weight.
10. Check Do Not Use For.
11. Pull existing location or generate a new one.

## Pull Guidance

- Prefer Tier 1 and Tier 2 locations for ordinary story use.
- Use Book One Risk or Protected Mystery Risk locations only when the scene truly needs that pressure.
- Do not use a sandbox location to redraw route topology or define faction law without review.
- Keep protected objects, signals, and false reports local unless the user explicitly approves canon review.

## Response Format for Pull Requests

When asked to pull a location, return:

```text
Best match
Backup option
Reason for fit
Canon safety note
How to introduce it
How it helps
How it complicates
Visual hook
```

## Final Principle

A location is useful before it is important.
