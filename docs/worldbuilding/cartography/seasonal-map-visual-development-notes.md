## Purpose

**Development Apocrypha**

This document records visual-development discoveries from generating four seasonal composite map concepts:

1. Dry Season
2. Wet Season
3. Cataract Season
4. Haboon Season

The maps were experiments that overlaid the three major cartographic traditions:

- Meridian / Human Traverse logistics mapping
- Antisapien / Hydropolis signal mapping
- Aeonolacertian / Craton pressure mapping

The first generated set used dramatic city / landmark scale for visual impact.

The second generated set corrected toward regional scale, making cities and landmarks smaller relative to the whole map.

These notes preserve what should be learned from that process.

---

## Primary Discovery: Scale Discipline

**Candidate Canon Seed**

Regional Meridian Prime maps should not exaggerate settlement and landmark size unless the map is explicitly symbolic, sacred, propaganda-based, tourist-facing, or dramatic.

### Correct Regional Scale Principle

On a true regional atlas map:

- Prime Ops should be a small highland logistics node, not a dominant city mass.
- Hydropolis should be a small city footprint / marker within the Mirror Basin, even if visually important.
- The four Hydropolis mooring towers should be visible as tiny vertical signatures or special symbols, not skyline-sized monuments dominating the basin.
- The Anticlinal Craton should be a major terrain form but should not swallow half the map.
- World Maw should be a dangerous landmark / pressure feature near the Craton sphere, not a continent-sized hole unless shown on a symbolic pressure map.
- The Dry Circuit should read as a long regional route system, not a short triangle between giant icons.

### Visual Rule

**The route must feel longer than the icons are large.**

## First Generated Map Deviations

**Visual Development Notes**

The first dramatic seasonal maps were useful for mood and map-language, but deviated from sandbox geography in several ways.

### Scale Deviations

- Hydropolis was visually too large relative to the Mirror Basin.
- The four towers read as skyline-dominant rather than small tower markers on a regional atlas.
- The Craton and World Maw became too visually dominant.
- The Dry Circuit felt like a simple symbolic triangle rather than a long seasonal route following river corridors, paleochannels, and hazard entrances.

### Geographic Deviations

- Some maps underemphasized the 8,000-foot descent as the required beginning of the Dry Circuit.
- Some maps made the route feel too direct instead of following dry riverbeds and seasonal corridors.
- The Craton sometimes read as always massively connected to the Maw rather than a distinct mid-elevation geode landform with nearby sacred / pressure geography.

### Useful Elements to Preserve

- The three-map overlay legend worked well: Meridian / Signal / Pressure.
- Seasonal title panels were strong.
- Water-state legends were useful.
- Signal overlays around Hydropolis were visually distinctive.
- Pressure rings and threshold fields around the Craton made the Aeonolacertian map logic legible.
- The idea of each season having a unique route-status label was strong.

## Second Generated Map Discoveries

**Candidate Canon Seeds**

The more scale-correct maps produced several useful atlas principles.

### 1. The Dry Circuit Should Be a Loop, Not a Triangle

The phrase **Dry Circuit** can still refer to the triangular civilization relationship, but the actual map route should read as a long, practical loop or circuit that:

- descends from the escarpment
- follows dry or intermittent channels
- uses paleochannel corridors
- approaches Hydropolis through basin routes
- loops toward the Craton approach / threshold areas
- returns toward the ascent / Prime Ops sphere

### 2. Waypoints Should Carry the Route's Rhythm

Numbered waypoints made the route feel operational.

Useful waypoint categories:

- start / end point
- descent gate
- switchback camp
- basin break camp
- Hydropolis approach
- channel crossing
- Craton spur camp
- threshold lookout
- ascent gate
- return camp

Future maps should include enough waypoints to imply end-of-shift intervals and route support without overcrowding the map.

### 3. Season Should Change the Route Type, Not Just the Color

The same corridor should transform across maps:

| Season | Route Behavior |
|---|---|
| Dry | active Traverse road / exposed riverbed route |
| Cataract | compromised evacuation route / braided flood hazard |
| Haboon | broken, mostly impassable storm route |
| Wet | submerged route / maritime or barge lane |

### 4. Hydropolis Needs a Seasonal Icon System

Hydropolis should not be redrawn wildly larger in wet season.

Instead, its icon should change state:

- dry season: grounded basin-floor city marker
- cataract season: rising / evacuation-watch marker
- haboon season: storm-mode marker with turbulent signal rings
- wet season: moored / floating city marker with water-lane connections

### 5. Craton Needs a Mountain / Island State Toggle

The Craton should have a consistent footprint but a different water relationship:

- dry season: geode mountain with land approach routes
- cataract season: mountain with lower approaches flooding
- haboon season: half-isolated geode mountain amid storm surge and floodsheet
- wet season: geode island with maritime / water approach logic

### 6. World Maw Should Be Marked, Not Magnified

World Maw is visually tempting to dramatize, but regional maps should mark it as:

- sacred wound
- pressure well
- vortex / submerged vortex in wet state if appropriate
- redacted / threshold-adjacent hazard
- pressure-field focus

It should not dominate the scale unless the map is specifically a World Maw pressure chart.

## Seasonal Route Status Labels

**Candidate Canon Seed**

Each seasonal map should include a route-status label.

Suggested labels:

| Season | Suggested Route Label |
|---|---|
| Dry | **Dry Circuit: Active Traverse Route** |
| Cataract | **Dry Circuit: Severely Compromised / Evacuate by Season's End** |
| Haboon | **Dry Circuit: Broken / All Routes Extreme Risk** |
| Wet | **Dry Circuit: Submerged / Maritime Lanes Only** |

These labels help preserve the concept that the Dry Circuit is not always a road.

## Seasonal Water-State Legends

**Candidate Canon Seed**

The generated maps proved that each season needs its own water-state legend.

### Dry Season Legend Needs

- exposed / dry
- intermittent / puddled
- subsurface / reduced
- ancient / paleochannel
- wet-season extent

### Cataract Season Legend Needs

- active cataract / falls
- rushing channel / flow
- flooded basin / early inland sea
- unstable ground
- route washout
- rising water extent
- evacuation route

### Haboon Season Legend Needs

- deep floodsheet / inland sea
- fast sheetflow / surge
- stormwater channel
- shallow / turbid flats
- subsurface / pressurized
- dust wall / haboob front
- washed / drifted / mobile terrain

### Wet Season Legend Needs

- deep water
- shallow water
- submerged silt flats
- seasonal ice shelf / permafrost edge
- maritime route
- barge lane
- submerged route closed
- mooring / port point

## Layering Lessons

**Development Apocrypha**

### Meridian / Human Traverse Layer

Works best when shown through:

- waypoint numbering
- route lines
- camps
- support nodes
- elevation rulers
- distance tables
- descent / ascent gates
- route status legends

### Antisapien / Hydropolis Signal Layer

Works best when shown through:

- cyan / teal subsurface traces
- purple contradiction regions
- RF quiet zones
- dead-channel belts
- identity seal markers
- signal-damping regions
- degraded / scattered signal notes in storm seasons

### Aeonolacertian / Craton Pressure Layer

Works best when shown through:

- red / amber guest-right boundaries
- pressure silence fields
- threshold lines
- route-memory veins
- sacred wound symbols
- Craton threshold zones
- pressure rise / siphon / silence labels

### Composite Rule

Each map should let one layer dominate according to use case, but all three should remain visible in composite atlas plates.

## Seasonal Visual Identity

**Candidate Canon Seed**

### Dry Season

Visual identity:

- pale exposed basin
- tan / cream crystal flats
- dry riverbeds
- highland crystal desert
- crisp route lines
- active camps
- signal traces visible but controlled

Emotional read:

**The world permits roadhood.**

### Cataract Season

Visual identity:

- retreating snowline
- waterfalls from escarpment
- braided violent rivers
- route breaks
- rising basin water
- warning symbols
- evacuation logic

Emotional read:

**The road is being taken back by water.**

### Haboon Season

Visual identity:

- dark storm palette
- dust walls
- floodsheet water
- lightning / static
- muddy surge
- broken route marks
- degraded signal marks
- unstable pressure boundaries

Emotional read:

**The map cannot keep up with the weather.**

### Wet Season

Visual identity:

- blue flooded basin
- snowfield escarpment
- submerged dry roads
- maritime lanes
- Hydropolis moored / floating
- Craton island
- depth contours

Emotional read:

**The road has become memory under water.**
