## Purpose

**Development Apocrypha**

This file collects reusable map symbols for Meridian Prime cartography.

It is a sandbox dictionary for later visual development, comic maps, game UI, tabletop handouts, in-world documents, and atlas work.

Symbols here are not automatically main canon.

They are candidate visual grammar.

---

## Symbol Design Principle

**Candidate Canon Seed**

A Meridian Prime map symbol should communicate four things when possible:

1. what the thing is
2. who recognizes it
3. what danger or permission it implies
4. what the mapmaker refuses to say directly
