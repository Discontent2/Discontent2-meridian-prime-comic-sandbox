## Purpose

World Works rarely calls a dangerous phenomenon by its true human, sacred, or civic name.

This file collects sanitized, bureaucratic, corporate, and WWISSaRD-style euphemisms for physics-adjacent anomalies.

Core principle:

**World Works does not lie by saying nothing exists. It lies by naming the thing until nobody can feel it anymore.**

---

## Usage Rules

Use euphemisms to:

- hide moral stakes
- reduce people to assets
- turn sacred thresholds into access zones
- turn living risks into liability categories
- confuse field crews
- create plausible deniability
- let WWISSaRD overclaim uncertain physics
- make corporate documents sound clean while describing monstrous things

Do not use this file to solve protected mysteries.

## How To Write World Works Physics Prose

World Works prose should be:

- clean
- abstract
- liability-aware
- morally refrigerated
- slightly too confident
- allergic to sacred language
- eager to say access instead of permission
- eager to say interface instead of person
- eager to say asset instead of guest
- eager to say artifact instead of warning

Bad corporate sentence:

**We found a dangerous alien doorway and want to own it.**

Better corporate sentence:

**The survey area presents non-Hydropolis access resilience potential pending geologic-structural validation.**

## WWISSaRD Overclaim Pattern

WWISSaRD may overclaim uncertain physics with phrases like:

- preliminary correlation suggests
- candidate interface behavior
- not inconsistent with gateway-adjacent models
- requires further containment review
- may represent non-equilibrium access persistence
- should not be dismissed as purely geologic
- retains strategic relevance despite ambiguous signal return

Translation:

**We do not know what this is, but we can get funding by implying it is a door.**
