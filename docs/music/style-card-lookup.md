# Meridian Prime Style Card Lookup

**Status:** Sandbox Music Reference / Non-Canon Until Promoted  
**Repository:** `Discontent2/Discontent2-meridian-prime-comic-sandbox`  
**Purpose:** Search metadata index for Meridian Prime music style cards.

---

## Why This Exists

Some band identities are embedded inside individual style cards, which can make broad lookup difficult. This file gives every known style card a compact searchable metadata block without requiring destructive rewrites of older style-card files.

Use this file to answer:

```text
what style cards exist
what band uses this style
which style card belongs to Gentrification
which style card belongs to Mergenta
which style card belongs to Time Bastard
which style card belongs to The Snifters
which style card belongs to Dead Mile Cipher
which style card belongs to Gloomed Vug
Meridian Prime style card lookup
```
