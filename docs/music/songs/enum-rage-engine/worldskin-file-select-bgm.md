## BGM Safety Note

This cue must sound like file-select and save-screen atmosphere, not interface information.

Do not use menu beeps, UI chirps, cursor sounds, select sounds, save sounds, save confirmation sounds, start-game sounds, button sounds, coin sounds, power-up sounds, alarms, sirens, lasers, or event cues.
