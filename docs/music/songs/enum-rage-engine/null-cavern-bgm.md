## Usage Note

`Null Cavern BGM` is Enum Rage Engine's first single and the first saved track to use the background-music-safe Enum prompt strategy.

This track is designed as background music, not a sound-effect-heavy arcade song. It should create area mood without confusing gameplay cues.
