## Core Concept

```text
The map does not show where you can go. It shows which wounds are still open.
```

Expanded concept:

```text
The biome-select screen is a cartographic threat display pretending to be helpful. Eight biome nodes rotate under green glass, each giving only a broken motif fragment before the player chooses a wound.
```
