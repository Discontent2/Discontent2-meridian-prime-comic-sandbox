## BGM Safety Note

This cue must sound like failure-screen and continue-screen atmosphere, not interface information or a literal arcade failure signal.

Do not use game-over jingles, fail sounds, death sounds, death stings, continue beeps, countdown sounds, timer sounds, menu beeps, UI chirps, cursor sounds, select sounds, start-game sounds, button sounds, coin sounds, power-up sounds, alarms, sirens, lasers, or event cues.

## Core Concept

```text
The game does not ask if you want to continue. It asks what part of you is left to spend.
```

Expanded concept:

```text
The failure screen is not a jingle or a punishment flourish. It is the cartridge lowering the lights and counting the player's remaining shape.
```
