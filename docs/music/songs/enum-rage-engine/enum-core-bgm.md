## BGM Safety Note

This cue must sound like atmosphere, pressure, and final-level dread, not gameplay information.

Do not use alarm tones, sirens, lock-on sounds, scan sounds, laser zaps, UI chirps, menu beeps, power-up sounds, door sounds, weapon sounds, footsteps, creature noises, voice samples, choir, or event cues.

```text

```
