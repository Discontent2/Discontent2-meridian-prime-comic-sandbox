## BGM Safety Note

This cue must sound like title-screen atmosphere, not interface information.

Do not use menu beeps, UI chirps, cursor sounds, select sounds, start-game sounds, button sounds, coin sounds, power-up sounds, alarms, sirens, lasers, or event cues.
