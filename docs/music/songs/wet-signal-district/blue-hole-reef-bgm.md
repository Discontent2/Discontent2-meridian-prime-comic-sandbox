## Canon / Continuity Note

`Blue Hole Reef BGM` is a working Worldskin game-continuity OST draft. It is treated as a Wet Signal District contribution to the Worldskin soundtrack lane.

It does not override mainline prose canon or define unrestricted truths about Project Green Glove, the Dry Gate, Conjugate systems, or àæonos.

This version exists to test a softer Pulse Width Music / Wet Signal District sound inside the 16-bit Worldskin OST format.

## Core Direction

```text
Wet Signal District got compressed into a haunted 16-bit underwater level.
```

This is not an Enum Rage Engine violence cue. It is Wet Signal District emotional place-music translated into console hardware.

## Core Concept

```text
a beautiful reef that remembers a train station
```

The track should feel like a tropical alien reef, an underwater cartridge level, and a sunken transit-station memory all at once.
