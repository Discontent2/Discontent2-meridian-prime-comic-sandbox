## Format Notes

### Gentrification Working Format

Use the `Tin Grin` format as the locked template for future Gentrification creature songs:

```text
short lines
ugly-funny images
direct slang hook
underproduced cassette Beach Goth
no lyric notes about instruments or vocal style
no real band names
no citations inside lyrics
```

### Time Bastard Studio Format

Use `Route to Hell-Wreck Deluxe` as the studio-album lane:

```text
studio track
no live crowd noise unless specifically requested
sleazy horror-boogie
short repeated hook
dirty direct road-ballad imagery
no real artist names
```
