## Core Identity

Program State Loop Deamon is a vocalist and founder of morosecodeclique.

He was originally part of the emo band Beast Jaw before becoming one of the figures who shaped morosecodeclique's Antisapian cyber-emo rap identity.

The spelling `Deamon` is intentional. It should feel like a corrupted process name rather than clean technical terminology.

Program State Loop Deamon turns heartbreak into a process that will not terminate.

Beast Jaw was the earlier emo band associated with Program State Loop Deamon before morosecodeclique.

Beast Jaw should feel like exposed-nerve Antisapian emo rock: guitar feedback, rehearsal-room sweat, bad amplifiers, torn flyers, and confession written too close to the mic.
