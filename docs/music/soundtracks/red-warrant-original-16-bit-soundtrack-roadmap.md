## Core Soundtrack Mission

`Red Warrant` needs tracks.

The soundtrack should deliver aggressive, playable, loop-safe 16-bit BGM for a fast side-scrolling combat platformer / beat-'em-up shooter hybrid.

This is not a band album first. It is game BGM first.

---

## Core Formula

```text
16-bit crossover-thrash combat BGM
hardcore pit timing
fast square-wave riff bass
aggressive FM guitar-like lead
fake chip kick
tiny digital snare tick
metallic FM hat
industrial chip sub pressure
short memorable riff cycles
mosh-breakdown combat sections
boss-room panic
```

---

## Pipeline

```text
Writ Hammer writes the riot.
Pit ROM loads it into the cartridge.
Devilment Studio turns it into stages.
```

---

## BGM Safety Rule

Core rule:

```text
If it sounds like gameplay information, keep it out of the BGM.
If it sounds like violence in the room, it can stay.
```

Avoid in soundtrack prompts:

```text
UI chirps
menu beeps
coin sounds
power-up sounds
item sounds
warning tones
alarms
sirens
lasers
realistic gunshots
impact sound effects
voice samples
spoken words
vocals
choir
realistic drums
modern hi-fi polish
real band names
direct imitation
```
