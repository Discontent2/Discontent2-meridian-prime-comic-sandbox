## Core Identity

Pulse Width Music is an Arrestthemall-adjacent electronic imprint founded by Enum Rage Engine.

It exists for electronic music that feels built from signal damage, waveform pressure, cartridge failure, dead CRT glow, and low-resolution voltage.

Core phrase:

```text
Pulse Width Music is where the square wave learned to breathe.
```

---

## Why the Name Works

`PWM` evokes pulse-width modulation, chip synthesis, square waves, old console hardware, oscillator logic, voltage control, and rhythmic machine pressure.

As `Pulse Width Music`, it also reads like a real imprint rather than a one-off joke.

Name meanings:

```text
Pulse = rhythm, voltage, body-machine motion
Width = waveform shape, modulation, signal control, room inside the beat
Music = label-legible, release-ready, less corporate than Records
PWM = hard acronym, hardware code, logo-friendly mark
```
