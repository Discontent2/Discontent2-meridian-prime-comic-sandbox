## Core Style

Enum Rage Engine creates primitive, ugly, fixed-loop Antisapian console music from degraded 16-bit hardware textures, industrial EBM pulse, and broken arcade-board aggression.

This is not bright retro nostalgia. It is a bootleg fight cartridge coughing through a tiny speaker.
