# Apocrypha

> Useful non-canon material that should remain available without being mistaken for canon.

**Sandbox Status:** Apocrypha Guide  
**Canon Status:** Not canon. Apocrypha may inspire future work but does not carry canon authority.  
**Authoritative Promotion Rules:** `Discontent2/meridian-prime:docs/development/sandbox-promotion-rules.md`

## Purpose

Apocrypha is creative compost.

It preserves material that is useful, atmospheric, contradictory, alternate, rejected, mythic, experimental, or adaptation-facing without letting it leak into canon by accident.

## What Belongs Here

Use apocrypha for:

- alternate traditions
- rejected-but-useful ideas
- experimental myths
- contradictory accounts
- side-story fragments
- unreliable documents
- adaptation-only concepts
- room experiments that should not become Candidate Canon yet
- legends that are useful because they disagree

## What Does Not Belong Here

Do not use apocrypha for:

- authoritative canon
- Candidate Canon staging
- protected mystery answers
- full canon summaries
- room onboarding guides
- promotion-ready proposals

## Recommended Subfolders

Create these only when needed:

```text
do-not-promote/
alternate-continuity/
contradictory-accounts/
experimental-myths/
side-story-fragments/
adaptation-only/
```

## Labeling Rule

Every apocrypha file should say why it is non-canon.

Recommended line:

```md
**Apocrypha Reason:** Alternate tradition / rejected-but-useful / experimental myth / contradictory account / adaptation-only / side-story fragment.
```

## Review Rule

Apocrypha may inspire future canon, but it must move through Candidate Canon review first.

Do not promote apocrypha directly into main canon.

## Core Rule

Not every good idea should become canon.

The sandbox should remain wild. Promotion should be disciplined.
