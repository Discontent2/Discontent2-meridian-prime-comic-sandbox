# Candidate Canon Intake

> Sandbox staging area for ideas that may deserve later main-canon review.

**Sandbox Status:** Candidate Canon Intake Guide  
**Canon Status:** Not canon. Material here is not canon unless explicitly promoted into `Discontent2/meridian-prime`.  
**Authoritative Promotion Rules:** `Discontent2/meridian-prime:docs/development/sandbox-promotion-rules.md`

## Purpose

This folder is for promotion-ready ideas only.

It is not for raw brainstorming, loose notes, full room transcripts, visual experiments, or unfinished lore piles.

Candidate Canon should be rare. An idea belongs here only when it strengthens existing canon, avoids protected mystery damage, and deserves deliberate review.

## Required Fields

Every Candidate Canon entry should include:

- Canon Weight
- Promotion Risk
- Primary Canon Dependencies
- Protected Mystery Contact
- Seasonal Impact
- Route / Jurisdiction Impact
- Story Use
- Recommended Review Path

## Recommended Review Paths

Use one of these:

- Hold in sandbox
- Test in sandbox stories or visuals
- Revise before review
- Ready for canon review
- Do not promote

## Folder Use

Room-specific subfolders may be created when needed:

```text
worldbuilding/
visual-development/
story/
analogy-room/
comic/
toy-development/
game-tabletop/
```

Do not create a subfolder until a room has promotion-ready material.

## Candidate Canon Checklist

Before placing an idea here, ask:

1. Is this more than just cool?
2. Does it improve existing canon without replacing it?
3. Does it avoid solving protected mysteries?
4. Does it have a clear canon weight?
5. Does it change routes, seasons, jurisdictions, species logic, story structure, or cosmology?
6. Is main-canon review actually needed?

## Core Rule

Candidate Canon is a waiting room, not a throne room.

The sandbox should remain wild. Promotion should be disciplined.
