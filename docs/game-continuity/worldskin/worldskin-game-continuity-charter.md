## Purpose

This file establishes the continuity rules for the gamified `Worldskin` project inside the Meridian Prime IP.

`Worldskin` is canon inside its own interactive/game continuity realm. It is not a direct literal transcript of mainline prose continuity, and it must not be used to reveal, simplify, or overwrite restricted lore about Dry Gate mechanics, Conjugate systems, àæonos ecosystems, or the full truth of Project Green Glove.

---

## Continuity Definition

```text
Worldskin is canon.
Worldskin is gamified canon.
Worldskin is not unrestricted mainline exposition.
```

The game continuity should be treated as an official Meridian Prime branch similar to an extended-universe / interactive adaptation lane:

```text
Mainline Meridian Prime Canon = core novels, bible, and primary lore
Interactive Continuity = official game-canon branch
Worldskin Game Realm = stylized playable interpretation of Project Green Glove / Worldskin logic
Worldskin OST = canon soundtrack artifact within that game branch
```

## Core Game-Realm Thesis

Each level is inside a Worldskin-contained biome patch.

The levels are not random lands. They are controlled contact ecologies: curated sample patches, test environments, containment theaters, and corporate lies wearing biome skin.

Core line:

```text
Every level is a lie wearing biome skin.
```

## Player-Facing Versus Internal Naming

Each level should have two names:

```text
Player-facing biome name = readable game level title
World Works designation = internal corporate test-patch label
```

Example:

```text
Null Cavern = player-facing level title
WS-01 Subterranean Contact Patch = World Works designation
```

This keeps the game playable while preserving the corporate containment logic underneath.

## BGM Continuity Rule

Enum Rage Engine's soundtrack is canon within the Worldskin Game Realm.

The soundtrack should remain background-music-safe:

```text
music-only background cue
non-interactive BGM
low-distraction area music
no gameplay cues
no event sounds
no voices
no realistic drums
```

For game clarity:

```text
If it sounds like information, keep it out of the BGM.
If it sounds like atmosphere, it can stay.
```
