# Room Outputs

> Sandbox home for outputs from specialized Meridian Prime rooms.

**Sandbox Status:** Workflow Guide  
**Canon Status:** Not canon. Room outputs are development material unless separately reviewed and promoted.  
**Authoritative Promotion Rules:** `Discontent2/meridian-prime:docs/development/sandbox-promotion-rules.md`

## Purpose

Room outputs should stay in room-specific folders so the sandbox remains useful instead of becoming a glitter storm in a filing cabinet.

This area is for organized outputs from rooms such as:

- Story / Novella Room
- World Building Department / Mythic Cosmology Lab
- Visual Development Room
- Comic Room
- Toy Development Room
- Analogy Room / Pattern Cartography Lab
- Game / Tabletop / IP Expansion rooms

## Recommended Folder Pattern

Create room folders only when needed:

```text
story-room/
worldbuilding/
visual-development/
comic-room/
toy-development/
analogy-room/
game-tabletop/
ip-expansion/
```

## Required Labels

Every room output should use one clear label:

- Experiment
- Candidate Canon Seed
- Apocrypha
- Visual Development
- Side Story
- Do Not Promote
- New IP Seed
- Handoff Note

## Output Rules

Room outputs should:

- name their source files
- separate canon from sandbox invention
- flag protected mystery contact
- avoid duplicating full canon summaries
- state intended use
- state whether review is needed

Room outputs should not:

- promote lore to canon
- replace source-of-truth files
- become giant master bibles
- mix unrelated rooms into one folder
- make Candidate Canon the default label

## Core Rule

Room folders are workshops. Canon is the archive.

The sandbox should remain wild. Promotion should be disciplined.
