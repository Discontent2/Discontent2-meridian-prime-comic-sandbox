# Addendum: Surreal Naturalist Gag Logic D20 Table

**Status:** Sandbox Media Artifact / Non-Canon / Requires Review Before Promotion  
**Project:** Meridian Prime  
**Newspaper:** The Anomalist  
**Funnies Section:** The Deviations  
**Humor Mode:** Surreal Naturalist Gag Logic

## Purpose

This addendum helps generate single-panel gag cartoon premises with a dry, surreal, newspaper-funnies sensibility built around:

- animals behaving like people
- people behaving like animals
- scientists missing the obvious
- ordinary institutions inside impossible situations
- deadpan captions
- cosmic accidents treated like daily errands
- nature having opinions
- civilization being very small and very confident

Do not name or imitate any real cartoonist. Use this as an original Meridian Prime house-humor mode.

## Core Principle

The joke should feel like nature, science, and bureaucracy all showed up to the same meeting and nobody brought the right body.

## When To Use

Use this addendum when the user wants:

- single-panel gag
- surreal naturalist humor
- weird animal cartoon
- science gag
- prehistoric gag
- lab gag
- talking creature gag
- deadpan newspaper cartoon
- The Deviations panel
- cosmic absurdity
- ordinary life in an impossible world

## Roll Format

Use up to:

```text
10 d20 rolls + 1 optional Wild Die
```

If the user provides fewer rolls, generate from the provided rolls.

If the user provides more than 10 rolls, use the first 10 as the core gag and treat roll 11 as the Wild Die unless the user says otherwise.
