# Card Templates

> Reusable sandbox templates for room outputs, candidate canon review, adaptation planning, and pattern work.

**Sandbox Status:** Prompt / Template Library  
**Canon Status:** Not canon. These templates organize work only.  
**Authoritative Promotion Rules:** `Discontent2/meridian-prime:docs/development/sandbox-promotion-rules.md`

## Use Rules

- Keep cards short.
- Do not copy whole canon files into cards.
- Label canon, sandbox, apocrypha, and Candidate Canon clearly.
- Include protected mystery warnings when relevant.
- Use source files for authority.
- Candidate Canon should be rare.
